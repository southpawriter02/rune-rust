# v0.9.2c Changelog: UI & Interaction Sounds

**Version:** 0.9.2c  
**Phase:** UI & Interaction Sounds

## Summary

Implements UI and interaction sound services with terrain-based footsteps, item sounds, puzzle feedback, and environment interactions.

---

## Added

### Application Layer - Interfaces

| Interface | Description |
|-----------|-------------|
| `IUISoundService` | UI sound triggering |
| `IInteractionSoundService` | Interaction sound triggering |

**IUISoundService Methods:**
- `PlayButtonClick()` — Button click
- `PlayButtonHover(volume)` — Button hover
- `PlayMenuOpen()` — Menu open
- `PlayMenuClose()` — Menu close
- `PlayNotification()` — Notification
- `PlayError()` — Error
- `PlayTabChanged()` — Tab change

**IInteractionSoundService Methods:**
- `PlayItemPickup()` — Item pickup
- `PlayItemDrop()` — Item drop
- `PlayItemEquip()` — Item equip
- `PlayItemUse()` — Item use
- `PlayGoldCollected()` — Gold coins
- `PlayPuzzleCorrect()` — Puzzle correct
- `PlayPuzzleIncorrect()` — Puzzle incorrect
- `PlayPuzzleComplete()` — Puzzle complete
- `PlayLeverPull()` — Lever pull
- `PlayFootstep(terrain)` — Terrain footstep
- `PlayDoorOpen()` — Door open
- `PlayDoorClose()` — Door close
- `PlayChestOpen()` — Chest open

### Infrastructure Layer - Services

| Class | Description |
|-------|-------------|
| `UISoundService` | UI sound implementation |
| `InteractionSoundService` | Interaction sound implementation |

### Terrain Footstep Mappings

| Input | Effect ID |
|-------|-----------|
| stone, dungeon, cave | footstep-stone |
| wood, floor, tavern | footstep-wood |
| grass, forest, nature | footstep-grass |
| water, shallow, marsh | footstep-water |
| metal, grate | footstep-metal |
| sand, desert | footstep-sand |
| (unknown) | footstep-stone |

- 4 UI effects (notification, error, menu open/close)

- 4 puzzle effects (correct, incorrect, complete, lever)
- 9 movement effects (6 footsteps + doors/chest)
