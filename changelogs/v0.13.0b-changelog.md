# v0.13.0b Changelog: AoE Zone Overlay

**Release Date:** 2026-01-20

---

## Summary

Implemented AoE Zone Overlay UI components for displaying active hazard zones, targeting previews, and zone legends on the combat grid.

---

## Added

### Presentation Layer

- **ZoneType** (enum) — Zone effect categorization
  - Damage, Control, Debuff, Buff, Preview with priority ordering

- **ZoneDisplayDto** — Data transfer object for zone rendering
  - Zone ID, name, type, affected cells, duration, damage type, effects

- **ZoneLegendEntryDto** — Data transfer object for legend entries
  - Symbol, name, duration, effect summary, color

- **ZoneDisplayConfig** — Configuration class with nested records
  - `ZoneSymbolConfig` — Zone type symbols
  - `ZoneColorConfig` — Zone type colors
  - `ZoneDisplaySettings` — Display behavior flags

- **ZoneRenderer** — Rendering logic for zones
  - Zone type to symbol/color mapping
  - Damage type to symbol/color mapping with fallbacks
  - Duration formatting (permanent, expiring, N turns)
  - Unicode/ASCII hazard icons
  - Cell content formatting

- **AoEZoneOverlay** — Main UI component
  - Zone rendering with cell-to-zone mapping
  - Priority-based overlap resolution (Damage > Control > Debuff > Buff > Preview)
  - Targeting preview support
  - Zone legend generation

### Configuration Files

- `config/zone-display.json` — Display configuration
- `config/schemas/zone-display.schema.json` — JSON Schema validation

### Tests

- `AoEZoneOverlayTests` (4 tests)
  - Multiple zones cell mapping
  - Overlapping zone priority
  - Preview mode highlighting
  - Legend formatting

- `ZoneRendererTests` (12 tests)
  - Zone symbol mapping
  - Damage type symbols (8 types)
  - Duration formatting
  - Hazard icons (Unicode/ASCII)
  - Cell formatting
