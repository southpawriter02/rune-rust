# v0.15.6d - Flora & Fauna Observation

**Release Date:** 2026-01-26

## Summary

Implements value objects and interfaces for the **Flora & Fauna Observation** system. Builds on existing `FloraFaunaDescriptor` entity and `FloraFaunaCategory` enum.

## Added

### Value Objects (Domain)

| Value Object               | Purpose                                         |
| -------------------------- | ----------------------------------------------- |
| `HarvestableFlora`         | Harvest DC, risk, and alchemical use info       |
| `SpeciesExaminationResult` | Species examination with layers and discoveries |

### Interfaces (Application)

| Interface                         | Purpose                                       |
| --------------------------------- | --------------------------------------------- |
| `IFloraFaunaDescriptorRepository` | Species descriptor data access                |
| `IPlayerDiscoveryTracker`         | Tracks discovered species per room            |
| `IFloraFaunaService`              | Flora/fauna observation, examination, harvest |

## Key Features

### HarvestableFlora VO

- Properties: `SpeciesId`, `SpeciesName`, `HarvestDc`, `HarvestRisk`, `AlchemicalUse`
- Helpers: `IsRisky`, `HasAlchemicalUse`, `ToDisplayString()`, `ToDetailedString()`

### SpeciesExaminationResult VO

- Properties: `SpeciesId`, `SpeciesName`, `ScientificName`, `Category`, `LayersRevealed`
- Layer tracking: `HighestLayerReached`, `SuccessCount`
- Discovery tracking: `AlchemicalUseRevealed`, `HarvestInfoRevealed`
- Helpers: `ReachedExpertLevel`, `GetDisplayHeader()`, `GetCombinedDescription()`
