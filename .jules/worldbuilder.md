# Worldbuilder Journal

## 2025-05-23 - Legacy Tag Conversion
**Design Constraint:** Legacy Entity "Tags" (e.g., "Brittle", "Undead") in Seeders lack mechanical definition for the d10 system.
**Solution:** Converted behavioral tags like "Brittle" into explicit Traits (e.g., "Weakness to Bludgeoning") within the new Entity Spec format.
