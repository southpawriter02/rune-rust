## Domain 8: Geography Constraints

### Canonical Ground Truth

**Realm-Specific Biomes:**

*   Each realm has documented biome characteristics
*   Gazetteer Collection = authoritative geographic source
*   Travel routes follow established infrastructure
*   Strategic locations consistent with factional control

**Geographic Features:**

*   **Midgard:** Greatwood, Asgardian Scar, Souring Mires, Serpent Fjords
*   **Vanaheim:** Canopy Sea, Gloom-Veil, Under-growth stratification
*   **Jötunheim:** Knuckle-Bone Fields, Ginnungagap Coolant Chasm, Ash-Blown Wastes
*   **\[Other realms\]:** Per Gazetteer Collection documentation

### Audit Checklist

*   \[ \] Biome descriptions match Gazetteer Collection?
*   \[ \] Geographic features consistent with realm characteristics?
*   \[ \] Travel routes logically positioned?
*   \[ \] Factional territorial control aligns with faction dossiers?
*   \[ \] Strategic locations referenced appropriately?
*   \[ \] No contradictions with established geography?
*   \[ \] Realm-specific hazards present where appropriate?

### Common Violations

❌ Geographic contradictions (misplaced biomes, impossible terrain)
❌ Travel routes inconsistent with documented infrastructure
❌ Factional control contradictions
❌ Unestablished locations without context

### Green Flags

✅ References to documented Gazetteer locations
✅ Biome characteristics consistent
✅ Travel infrastructure logical
✅ Factional control appropriate
