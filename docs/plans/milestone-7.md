# Milestone 7: The Architect (Settlements & Economy)

This phase shifts focus from dungeon crawling to the "Social Sector," establishing safe zones, commerce, and political interaction.

	•	[ ] v0.5.0: Settlement Engine
	◦	Scope: Implement the SettlementGenerator to create safe zones with distinct districts (Gate, Market, Residential) and wire the enter/exit commands for transitioning between dungeon and settlement sectors.
	•	[ ] v0.5.1: Trade & Merchants
	◦	Scope: Implement TradeService, Merchant entities, and the buy/sell commands, supporting dynamic stock generation and currency (Scrip) exchange.
	•	[ ] v0.5.2: Advanced Factions
	◦	Scope: Implement granular Reputation tiers (Hated to Exalted) and faction-specific service gating (e.g., Dvergr smiths refusing service to low-reputation players).
	•	[ ] v0.5.3: Quest Chains
	◦	Scope: Expand the QuestService to support multi-stage narratives, branching outcomes based on choices, and delivery missions between settlements.
	•	[ ] v0.5.4: The World Map
	◦	Scope: Implement WorldMapService and fast travel mechanics, introducing "Journey Mode" for calculating travel time and random encounters between settlements.
