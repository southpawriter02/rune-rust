## v0.13.0 - UI Consolidation Pass

**Focus:** TUI/GUI integration for features introduced in v0.10.x through v0.12.x

### Overview

This version consolidates all deferred UI work from the preceding feature versions into a single, cohesive UI pass. By batching UI implementation, we ensure consistent styling, reduce context-switching, and make holistic design decisions with full knowledge of all features requiring display.
