## Summary

Implements an end-of-combat summary window displaying victory/defeat status, rewards (XP, gold, loot), level-up notifications, and combat statistics with action buttons.

---

## Added

### Presentation Layer - ViewModels

| ViewModel | Description |
|-----------|-------------|
| `CombatSummaryViewModel` | Main ViewModel with commands |
| `CombatStatisticsViewModel` | Combat metrics display |
| `LootItemViewModel` | Loot item with quantity |
| `LevelUpBonusViewModel` | Level-up bonus display |

**CombatSummaryViewModel Properties:**
- `IsVictory` — Victory/defeat state
- `XpEarned`, `GoldEarned` — Rewards
- `DefeatedEnemies` — List with skull icons
- `Loot` — Collection of loot items
- `Statistics` — Combat metrics
- `DidLevelUp`, `NewLevel` — Level-up state
- `LevelUpBonuses` — Stat increases, abilities

**CombatSummaryViewModel Commands:**
- `CollectAll` — Collects loot (C key)
- `Continue` — Closes window (Enter key)
- `LoadSave` / `ReturnToMenu` / `Retry` — Defeat options

### Presentation Layer - Views

| View | Description |
|------|-------------|
| `CombatSummaryWindow.axaml` | Victory/defeat modal |

**Window Sections:**
- Victory/defeat header
- Enemies defeated list
- Rewards panel (XP, gold, loot)
- Level-up panel (conditional)
- Combat statistics (6 metrics)
- Action buttons

### Presentation Layer - Styles

| File | Description |
|------|-------------|
| `CombatSummaryStyles.axaml` | Summary window styling |

## Keyboard Shortcuts

| Key | Action | Context |
|-----|--------|---------|
| C | Collect all loot | Victory with loot |
| Enter | Continue | Victory |
| Escape | Close/Return | Both |
