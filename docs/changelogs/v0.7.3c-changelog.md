# v0.7.3c Changelog - Combat UI Components

## Added

### ViewModels
- **`CombatantEntryViewModel`**: Displays individual combatant with HP, status effects, and turn indicator
- **`StatusEffectViewModel`**: Represents status effect icon with tooltip
- **`CombatantListViewModel`**: Manages combatant list sorted by initiative
- **`CombatActionBarViewModel`**: Manages action bar buttons and turn state

### Controls
- **`CombatantListPanel`**: Shows all combatants in turn order with HP bars
- **`CombatActionBar`**: Provides action buttons (Attack, Move, Ability, Defend, Item, Wait, End Turn)

### Styles
- **`CombatUIStyles.axaml`**: Styles for combatant entries, status effects, action bar
