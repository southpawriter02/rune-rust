# v0.15.2g Changelog - Specialization Integration

**Release Date:** 2026-01-23
**Focus:** Gantry-Runner and Myrk-gengr specialization abilities
**Prerequisites:** v0.15.2a-f Complete (Climbing, Leaping, Fall Damage, Stealth Movement, Chase Sequences, Balance Checks)

---

## Summary

This release completes the v0.15.2 Acrobatics Skill Expansion by introducing specialization-specific abilities that enhance the base skill systems. Gantry-Runner abilities focus on acrobatic movement, while Myrk-gengr abilities enhance stealth and infiltration.

---

## Added

### Domain Layer

#### Enums
- **`SpecializationAbilityType`**: Activation types (Passive, Triggered, Reactive, Active)
- **`GantryRunnerAbility`**: 5 acrobatics abilities
  - `RoofRunner`: Reduce climbing stages by 1
  - `DeathDefyingLeap`: +10 ft maximum leap
  - `WallRun`: Vertical run combat action
  - `DoubleJump`: Reroll failed leap (1/day)
  - `Featherfall`: Auto-succeed Crash Landing DC ≤ 3
- **`MyrkengrAbility`**: 4 stealth abilities
  - `SlipIntoShadow`: Enter [Hidden] without action
  - `GhostlyForm`: Stay [Hidden] after attack (1/encounter)
  - `CloakTheParty`: +2d10 party Passive Stealth
  - `OneWithTheStatic`: Auto-[Hidden] in [Psychic Resonance]

#### Extension Classes
- **`GantryRunnerAbilityExtensions`**: Display names, types, descriptions, usage info
- **`MyrkengrAbilityExtensions`**: Display names, types, descriptions, usage info

#### Value Objects
- **`SpecializationAbilityDefinition`**: Ability metadata with factory methods
- **`AbilityActivationContext`**: Activation parameters and trigger info
- **`AbilityActivationResult`**: Outcome with effects and narrative

### Application Layer

#### Interfaces
- **`ISpecializationAbilityService`**: Full ability operations contract

#### Services
- **`SpecializationAbilityService`**: Complete implementation with:
  - Ability availability checking
  - Gantry-Runner ability activations
  - Myrk-gengr ability activations
  - Daily/encounter use tracking
  - Comprehensive logging

### Configuration

- **`config/specialization-abilities.json`**: All ability definitions
- **`config/schemas/specialization-abilities.schema.json`**: JSON Schema
