## Summary

Implements the player status panel displaying character info, HP/MP/XP bars with color thresholds, stats with modifiers, equipment slots, gold, and current location for the RightSidebarTop region.

## Panel Features

| Feature | Description |
|---------|-------------|
| Character Info | Name, class, level display with icon |
| Resource Bars | HP (color thresholds), MP, XP bars |
| Stats Grid | 6 attributes with D&D-style modifiers |
| Equipment | 5 slots (weapon, armor, shield, ring, amulet) |
| Footer | Gold amount and current location |

---

## HealthBarControl Features

| Property | Description |
|----------|-------------|
| `CurrentValue` | Current resource value |
| `MaxValue` | Maximum resource value |
| `BarType` | Health/Mana/XP/Stamina/Custom |
| `Label` | Bar label text |
| `FillPercentage` | Computed fill percentage |
| `FillBrush` | Color based on type and health thresholds |
