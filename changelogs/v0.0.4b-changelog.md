# v0.0.4b Changelog: Resource Pool System

**Release Date:** 2026-01-06
**Version:** 0.0.4b

## Overview

Implements class-specific resource pools with unique mechanics (regeneration, decay, combat build).

---

## Features

### Resource Types (6)
| Resource | Regen | Decay | Build On | Start |
|----------|-------|-------|----------|-------|
| Health | 0 | 0 | - | Max |
| Mana | +10/turn | 0 | - | Max |
| Rage | 0 | -10/turn | Deal/Take dmg | Zero |
| Energy | +20/turn | 0 | - | Max |
| Faith | +5/turn | 0 | Heal | Max |
| Focus | +8/turn | 0 | - | Max |

### Mechanics
- **Regeneration**: Mana, Energy, Focus, Faith restore per turn
- **Decay**: Rage decays out of combat only
- **Build-on-Hit**: Rage +10 dealt, +15 taken; Faith +15 heal
