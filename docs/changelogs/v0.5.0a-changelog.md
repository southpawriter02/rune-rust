# v0.5.0a Changelog: Grid Core

**Version:** 0.5.0a  
**Phase:** Grid Core  
**Date:** 2026-01-14  
**Status:** ✅ Complete

---

## Summary

Implements the foundational combat grid infrastructure for tactical positioning. This version establishes the core data structures and services needed for the Combat Grid system.

---

## Added

### Domain Layer

| Component | File | Description |
|-----------|------|-------------|
| `MovementDirection` | Enums/MovementDirection.cs | 8-direction enum (N, S, E, W, NE, NW, SE, SW) |
| `GridPosition` | ValueObjects/GridPosition.cs | X,Y coordinates with distance calculations and chess notation |
| `GridCell` | ValueObjects/GridCell.cs | Cell state tracking (occupancy, passability, display) |
| `CombatGrid` | Entities/CombatGrid.cs | 2D grid entity with entity placement and movement |

### Application Layer

| Component | File | Description |
|-----------|------|-------------|
| `ICombatGridService` | Interfaces/ICombatGridService.cs | Grid management interface |
| `CombatGridService` | Services/CombatGridService.cs | Grid creation and entity initialization |
| `GridSettings` | Interfaces/IConfigurationProvider.cs | Grid configuration record |
| `GridInitializationResult` | Interfaces/ICombatGridService.cs | Result record for initialization |

### Infrastructure Layer

| Component | File | Description |
|-----------|------|-------------|
| `GetGridSettings()` | JsonConfigurationProvider.cs | Grid configuration loading |

### Configuration

| File | Description |
|------|-------------|
| config/grid.json | Default grid settings (8x8) with room overrides |
| config/schemas/grid.schema.json | JSON Schema validation |
