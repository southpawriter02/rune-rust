## Summary

Implements the core sound effect service with category-based organization, randomized variations, and max simultaneous sound enforcement.

---

## Added

### Application Layer - Interfaces

| Interface | Description |
|-----------|-------------|
| `ISoundEffectService` | SFX playback operations |
| `ISoundEffectConfig` | SFX configuration lookup |

**ISoundEffectService Members:**
- `Play(effectId, volume)` — Play effect by ID
- `PlayRandom(category, volume)` — Random from category
- `StopAll()` — Stop all playbacks
- `SetVolume/GetVolume` — Effects channel volume
- `PreloadCategoryAsync` — Preload category
- `GetAvailableEffects/GetCategories` — List queries

### Application Layer - DTOs

| DTO | Description |
|-----|-------------|
| `SoundEffectDefinition` | Effect with files |
| `SoundEffectSettings` | Global settings |
| `SoundEffectsConfig` | Root config |
| `CategoryDefinition` | Category with effects |
| `EffectFileConfig` | Raw effect config |

### Infrastructure Layer - Services

| Class | Description |
|-------|-------------|
| `SoundEffectService` | SFX implementation |
| `SoundEffectConfig` | Config with defaults |

**Features:**
- Max simultaneous (8) enforcement
- Randomized file selection for variation
- Volume multiplier per effect
- Category-based organization

### Configuration

| File | Description |
|------|-------------|
| `sound-effects.json` | 6 categories, 20+ effects |

**Categories:** combat, ability, ui, items, puzzle, movement
