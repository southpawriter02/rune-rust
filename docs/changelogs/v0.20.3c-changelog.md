# v0.20.3c Changelog — Jötun-Reader Tier 3 & Capstone

## Added

### Domain — Enums

- `LoreRevelationType` enum (7 values) for Ancient Knowledge lore revelation categories
- `JotunTechnologyType` enum (8 values) for Voice of the Giants technology types

### Domain — Value Objects

- `LoreRevelation` — bonus lore record with quest hooks, related locations/figures
- `TechnologyEffect` — effect produced by activated Jötun technology
- `ActivatedTechnology` — activation record with duration tracking and deactivation

### Application — Services

- `JotunReaderTier3AbilityService` — Lore Keeper, Ancient Knowledge, Voice of the Giants
    - Lore Keeper: auto-succeeds Layer 3 for historical objects, +2 Lore Insight on first exam
    - Ancient Knowledge: 1 bonus lore (Expert) / 2 bonus lore (Master), +1 Insight per entry
    - Voice of the Giants: activates dormant technology, once per long rest, 5 AP + 8 Insight

### Configuration

- `specializations.json` — Tier 3 ("Master Scholar") and Tier 4 (Capstone) added to JotunReader

### Tests

- `JotunReaderTier3AbilityServiceTests` — 16 unit tests covering all abilities and prerequisites
