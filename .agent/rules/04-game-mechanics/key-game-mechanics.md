## Key Game Mechanics to Remember

| Mechanic | Note |
|----------|------|
| Corruption | Permanent, accumulates on death |
| Stress | Goes to 100, triggers Trauma Check |
| Runic Blight | Corruption with specific effects |
| Coherent vs Heretical | Two path types for specializations |
| Stamina | Universal action resource |
| Fury | Berserkr-specific resource |
