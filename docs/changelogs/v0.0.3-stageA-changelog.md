# v0.0.3 Stage A Changelog: Core Player Creation

**Release Date:** 2025-01-06
**Version:** 0.0.3-stageA

## Overview

This release implements the core character creation system, allowing players to create customized characters with races, backgrounds, and point-buy attributes.

## New Features

### Character Creation System
- **Name Validation**: 2-30 characters, letters/spaces/hyphens/apostrophes only
- **Race Selection**: 4 playable races (Human, Elf, Dwarf, Halfling) with unique attribute modifiers and racial traits
- **Background Selection**: 8 backgrounds (4 professions + 4 lineages) with attribute bonuses and starter abilities
- **Point-Buy Attributes**: 25 points to allocate across 5 attributes (Might, Fortitude, Will, Wits, Finesse)
- **Character Description**: Optional 500-character backstory
- **Character Summary**: Review and confirm before starting

### Data-Driven Configuration
- JSON-based race, background, and attribute definitions
- Easy addition of new races/backgrounds without code changes
- Configurable point-buy rules (starting points, min/max values)
