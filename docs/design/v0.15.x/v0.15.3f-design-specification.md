## 1. Overview

### 1.1 Purpose

This document provides a comprehensive design specification for v0.15.3f, the Interrogation System phase. This subsystem implements multi-round information extraction mechanics where the interrogator attempts to break down a subject's resistance to obtain information. Unlike single-check social interactions, interrogation is an extended skill challenge that unfolds over multiple rounds with varying methods, reliability, and consequences. The key components are:

1. **InterrogationMethod Enum**: The five approaches available for extracting information: GoodCop (empathetic), BadCop (aggressive), Deception (trickery), Bribery (incentives), and Torture (extreme, with severe consequences).

2. **InterrogationStatus Enum**: Represents the current phase of interrogation from InProgress through to SubjectBroken, SubjectResisting, or Abandoned.

3. **SubjectResistance Enum**: Categorizes how many successful checks are required to break a subject's will, from Minimal (1 check) to Extreme (10+ checks).

4. **InterrogationState Entity**: Tracks the complete state of an interrogation session including subject WILL, resistance remaining, methods used, information gained, and disposition changes.

5. **InformationGained Value Object**: Records extracted information along with its reliability rating based on the method used.

6. **InterrogationService**: Orchestrates the multi-round interrogation process, managing resistance depletion, method selection, and information reliability.

This subsystem creates morally complex gameplay where players must weigh effective methods against ethical consequences, reliability of obtained information, and lasting relationship damage.

### 1.2 Current vs. Target Implementation

| Aspect | Current Implementation | Target Implementation |
|--------|------------------------|----------------------|
| **Resistance Tracking** | Not implemented | WILL-based resistance that depletes over rounds |
| **Multi-Round** | Single check | Extended skill challenge with multiple rounds |
| **Methods** | Fixed approach | Choose GoodCop, BadCop, Deception, Bribery, or Torture |
| **Resistance Depletion** | Not tracked | Success reduces resistance by 1 |
| **Information Reliability** | Not tracked | 50-95% reliability based on method used |
| **Subject Breaking** | Not implemented | Subject breaks when resistance reaches 0 |
| **Method Consequences** | Not tracked | Disposition changes, reputation costs, trauma |
| **Session History** | Not tracked | Complete history of all rounds and methods |
| **Torture Consequences** | Not implemented | -30 reputation, 50% reliability, trauma inflicted |

### 1.3 Scope

**In Scope:**
- Define `InterrogationMethod` enum with 5 methods
- Define `InterrogationStatus` enum with 5 statuses
- Define `SubjectResistance` enum with 5 resistance levels
- Define `InterrogationState` entity with session management
- Define `InterrogationRound` value object for round tracking
- Define `InterrogationContext` value object for round input
- Define `InterrogationResult` value object for outcomes
- Define `InformationGained` value object for extracted data
- Create `IInterrogationService` interface for interrogation operations
- Implement `InterrogationService` with multi-round logic
- Implement [Subject Broken] fumble consequence
- Configuration file `interrogation-config.json`
- Unit tests for interrogation mechanics (~3 tests)

**Out of Scope:**
- Cultural protocols - v0.15.3g
- Extended influence - v0.15.3h
- Specialization integration (Kupmaðr abilities) - v0.15.3i
- NPC interrogation of player characters

```csharp
// Location: src/Domain/Entities/InterrogationState.cs

namespace RuneRust.Domain.Entities;

using RuneRust.Domain.Enums;
using RuneRust.Domain.ValueObjects;

/// <summary>
/// Tracks the complete state of an interrogation session including
/// subject resistance, methods used, and information extracted.
/// </summary>
public sealed class InterrogationState
{
    private readonly List<InterrogationRound> _history = new();
    private readonly List<InformationGained> _informationGained = new();
    private readonly List<InterrogationMethod> _methodsUsed = new();

    /// <summary>
    /// Unique identifier for this interrogation session.
    /// </summary>
    public required string InterrogationId { get; init; }

    /// <summary>
    /// The character conducting the interrogation.
    /// </summary>
    public required string InterrogatorId { get; init; }

    /// <summary>
    /// The NPC being interrogated.
    /// </summary>
    public required string SubjectId { get; init; }

    /// <summary>
    /// The subject's WILL attribute value.
    /// Used to calculate base resistance and Torture DC.
    /// </summary>
    public required int SubjectWill { get; init; }

    /// <summary>
    /// The number of successful checks still required to break the subject.
    /// Starts at a value determined by SubjectWill and modifiers.
    /// Reaches 0 when subject breaks.
    /// </summary>
    public int ResistanceRemaining { get; private set; }

    /// <summary>
    /// The initial resistance value before any checks.
    /// </summary>
    public required int InitialResistance { get; init; }

    /// <summary>
    /// The resistance category of this subject.
    /// </summary>
    public required SubjectResistance ResistanceLevel { get; init; }

    /// <summary>
    /// The current round number (1-based).
    /// </summary>
    public int RoundNumber { get; private set; }

    /// <summary>
    /// The current status of the interrogation.
    /// </summary>
    public InterrogationStatus Status { get; private set; }

    /// <summary>
    /// Cumulative disposition change from methods used.
    /// Negative values indicate relationship damage.
    /// </summary>
    public int TotalDispositionChange { get; private set; }

    /// <summary>
    /// Cumulative reputation cost (primarily from Torture).
    /// </summary>
    public int TotalReputationCost { get; private set; }

    /// <summary>
    /// Cumulative resource cost (from Bribery).
    /// </summary>
    public int TotalResourceCost { get; private set; }

    /// <summary>
    /// Whether Torture has been used at any point.
    /// This caps maximum reliability at 60%.
    /// </summary>
    public bool TortureUsed { get; private set; }

    /// <summary>
    /// Read-only list of methods used (may contain duplicates).
    /// </summary>
    public IReadOnlyList<InterrogationMethod> MethodsUsed => _methodsUsed.AsReadOnly();

    /// <summary>
    /// Read-only access to the round history.
    /// </summary>
    public IReadOnlyList<InterrogationRound> History => _history.AsReadOnly();

    /// <summary>
    /// Read-only access to information gained.
    /// </summary>
    public IReadOnlyList<InformationGained> InformationGained => _informationGained.AsReadOnly();

    /// <summary>
    /// Initializes the interrogation state with starting values.
    /// </summary>
    public void Initialize()
    {
        ResistanceRemaining = InitialResistance;
        RoundNumber = 0;
        Status = InterrogationStatus.NotStarted;
        TotalDispositionChange = 0;
        TotalReputationCost = 0;
        TotalResourceCost = 0;
        TortureUsed = false;
        _history.Clear();
        _informationGained.Clear();
        _methodsUsed.Clear();
    }

    /// <summary>
    /// Records a completed round and updates state accordingly.
    /// </summary>
    public void RecordRound(InterrogationRound round)
    {
        if (Status.IsTerminal())
        {
            throw new InvalidOperationException("Cannot record rounds on a terminal interrogation.");
        }

        RoundNumber++;
        _history.Add(round);
        _methodsUsed.Add(round.MethodUsed);

        // Update status if this is the first round
        if (Status == InterrogationStatus.NotStarted)
        {
            Status = InterrogationStatus.InProgress;
        }

        // Apply resistance change
        ResistanceRemaining += round.ResistanceChange; // ResistanceChange is negative on success
        if (ResistanceRemaining < 0)
        {
            ResistanceRemaining = 0;
        }

        // Apply cumulative effects
        TotalDispositionChange += round.DispositionChange;
        TotalResourceCost += round.ResourceCost;

        // Track Torture usage
        if (round.MethodUsed == InterrogationMethod.Torture)
        {
            TortureUsed = true;
            TotalReputationCost += InterrogationMethod.Torture.GetReputationCost();
        }

        // Check for subject broken
        if (ResistanceRemaining == 0)
        {
            Status = InterrogationStatus.SubjectBroken;
        }
    }

    /// <summary>
    /// Adds extracted information to the session.
    /// </summary>
    public void AddInformation(InformationGained info)
    {
        _informationGained.Add(info);
    }

    /// <summary>
    /// Abandons the interrogation.
    /// </summary>
    public void Abandon()
    {
        if (!Status.IsTerminal())
        {
            Status = InterrogationStatus.Abandoned;
        }
    }

    /// <summary>
    /// Marks the subject as successfully resisting (optional max rounds).
    /// </summary>
    public void MarkSubjectResisting()
    {
        if (!Status.IsTerminal())
        {
            Status = InterrogationStatus.SubjectResisting;
        }
    }

    /// <summary>
    /// Gets the most frequently used method (primary method).
    /// Used to determine information reliability.
    /// </summary>
    public InterrogationMethod GetPrimaryMethod()
    {
        if (_methodsUsed.Count == 0)
        {
            return InterrogationMethod.GoodCop; // Default
        }

        return _methodsUsed
            .GroupBy(m => m)
            .OrderByDescending(g => g.Count())
            .First()
            .Key;
    }

    /// <summary>
    /// Calculates the reliability percentage based on methods used.
    /// </summary>
    public int CalculateReliability()
    {
        if (_methodsUsed.Count == 0)
        {
            return 0;
        }

        // If Torture was used at any point, cap reliability at 60%
        if (TortureUsed)
        {
            return Math.Min(60, GetPrimaryMethod().GetReliabilityPercent());
        }

        // Get primary method reliability
        var primaryMethod = GetPrimaryMethod();
        return primaryMethod.GetReliabilityPercent();
    }
}
```

## 13. InterrogationService Implementation

### 13.1 Definition

```csharp
// Location: src/Application/Services/InterrogationService.cs

namespace RuneRust.Application.Services;

using RuneRust.Application.Interfaces;
using RuneRust.Domain.Entities;
using RuneRust.Domain.Enums;
using RuneRust.Domain.Extensions;
using RuneRust.Domain.ValueObjects;

/// <summary>
/// Implements multi-round interrogation mechanics including resistance
/// depletion, method-based reliability, and consequence tracking.
/// </summary>
public sealed class InterrogationService : IInterrogationService
{
    private readonly IDiceRollerService _diceRoller;
    private readonly IPersuasionService _persuasionService;
    private readonly IDeceptionService _deceptionService;
    private readonly IIntimidationService _intimidationService;
    private readonly IFumbleConsequenceService _fumbleService;
    private readonly Random _random;

    public InterrogationService(
        IDiceRollerService diceRoller,
        IPersuasionService persuasionService,
        IDeceptionService deceptionService,
        IIntimidationService intimidationService,
        IFumbleConsequenceService fumbleService)
    {
        _diceRoller = diceRoller;
        _persuasionService = persuasionService;
        _deceptionService = deceptionService;
        _intimidationService = intimidationService;
        _fumbleService = fumbleService;
        _random = new Random();
    }

    /// <inheritdoc />
    public InterrogationState InitializeInterrogation(
        string interrogatorId,
        string subjectId,
        int subjectWill,
        int resistanceModifiers = 0)
    {
        var resistanceLevel = AssessResistance(subjectWill, resistanceModifiers);
        var baseResistance = CalculateInitialResistance(resistanceLevel, subjectWill, resistanceModifiers);

        var state = new InterrogationState
        {
            InterrogationId = Guid.NewGuid().ToString(),
            InterrogatorId = interrogatorId,
            SubjectId = subjectId,
            SubjectWill = subjectWill,
            InitialResistance = baseResistance,
            ResistanceLevel = resistanceLevel
        };

        state.Initialize();
        return state;
    }

    /// <inheritdoc />
    public SubjectResistance AssessResistance(int subjectWill, int resistanceModifiers = 0)
    {
        var effectiveWill = subjectWill + (resistanceModifiers / 2); // Modifiers contribute half
        return SubjectResistanceExtensions.FromWillAttribute(effectiveWill);
    }

    /// <inheritdoc />
    public InterrogationRound ConductRound(InterrogationContext context)
    {
        var state = context.InterrogationState;
        var method = context.SelectedMethod;

        if (state.Status.IsTerminal())
        {
            throw new InvalidOperationException("Cannot conduct rounds on a terminal interrogation.");
        }

        // Calculate DC
        var dc = CalculateDc(context);

        // Calculate dice pool
        var dicePool = CalculateDicePool(context);

        // Roll
        var rollResult = _diceRoller.Roll(dicePool);
        var successesRequired = (int)Math.Ceiling(dc / 4.0);
        var isSuccess = rollResult.NetSuccesses >= successesRequired;
        var isFumble = rollResult.IsFumble;

        // Calculate effects
        var resistanceChange = isSuccess ? -1 : 0;
        var dispositionChange = method.GetDispositionChangePerRound();
        var resourceCost = method.RequiresResources()
            ? CalculateBriberyCost(state.ResistanceLevel)
            : 0;

        // Determine outcome
        var outcome = DetermineOutcome(rollResult, successesRequired);

        // Handle fumble
        FumbleType? fumbleType = null;
        if (isFumble)
        {
            fumbleType = GetFumbleTypeForMethod(method);
        }

        // Generate narrative
        var narrative = GenerateRoundNarrative(context, isSuccess, isFumble, method);

        var round = new InterrogationRound
        {
            RoundNumber = state.RoundNumber + 1,
            MethodUsed = method,
            CheckResult = outcome,
            DiceRolled = dicePool,
            SuccessesAchieved = rollResult.NetSuccesses,
            SuccessesRequired = successesRequired,
            ResistanceChange = resistanceChange,
            ResistanceAfter = Math.Max(0, state.ResistanceRemaining + resistanceChange),
            DispositionChange = dispositionChange,
            ResourceCost = resourceCost,
            TimeElapsedMinutes = method.GetRoundDurationMinutes(),
            IsFumble = isFumble,
            FumbleType = fumbleType,
            NarrativeDescription = narrative
        };

        // Record the round
        state.RecordRound(round);

        return round;
    }

    /// <inheritdoc />
    public int CalculateBriberyCost(SubjectResistance resistanceLevel)
    {
        var baseCost = resistanceLevel.GetBaseBriberyCost();
        // Add some variance (±20%)
        var variance = (int)(baseCost * 0.2);
        return baseCost + _random.Next(-variance, variance + 1);
    }

    /// <inheritdoc />
    public int CalculateTortureDc(int subjectWill, int modifiers = 0)
    {
        return (subjectWill * 2) + modifiers;
    }

    /// <inheritdoc />
    public IReadOnlyList<InformationGained> ExtractInformation(
        InterrogationState state,
        IReadOnlyList<string> topics)
    {
        if (state.Status != InterrogationStatus.SubjectBroken)
        {
            throw new InvalidOperationException("Cannot extract information from unbroken subject.");
        }

        var reliability = state.CalculateReliability();
        var primaryMethod = state.GetPrimaryMethod();
        var result = new List<InformationGained>();

        foreach (var topic in topics)
        {
            // Determine if this specific piece of info is true
            var roll = _random.Next(1, 101);
            var isTrue = roll <= reliability;

            var content = GenerateInformationContent(topic, isTrue);

            var info = new InformationGained
            {
                Topic = topic,
                Content = content,
                ReliabilityPercent = reliability,
                SourceMethod = primaryMethod,
                IsVerified = null, // Not yet verified
                IsTrue = isTrue   // Hidden from player until verified
            };

            state.AddInformation(info);
            result.Add(info);
        }

        return result.AsReadOnly();
    }

    /// <inheritdoc />
    public InterrogationResult CompleteInterrogation(InterrogationState state)
    {
        var outcome = state.Status.IsSuccess()
            ? SkillOutcome.Success
            : SkillOutcome.Failure;

        var totalTime = state.History.Sum(r => r.TimeElapsedMinutes);

        return new InterrogationResult
        {
            Outcome = outcome,
            FinalStatus = state.Status,
            RoundsUsed = state.RoundNumber,
            TotalTimeMinutes = totalTime,
            PrimaryMethod = state.GetPrimaryMethod(),
            InformationReliability = state.CalculateReliability(),
            InformationGained = state.InformationGained,
            TotalDispositionChange = state.TotalDispositionChange,
            TotalReputationCost = state.TotalReputationCost,
            TotalResourceCost = state.TotalResourceCost,
            TortureWasUsed = state.TortureUsed,
            SubjectTraumatized = state.TortureUsed,
            History = state.History,
            NarrativeSummary = GenerateInterrogationSummary(state)
        };
    }

    /// <inheritdoc />
    public void AbandonInterrogation(InterrogationState state)
    {
        state.Abandon();
    }

    /// <inheritdoc />
    public IReadOnlyDictionary<InterrogationMethod, bool> GetAvailableMethods(
        InterrogationState state,
        int availableGold)
    {
        var result = new Dictionary<InterrogationMethod, bool>
        {
            [InterrogationMethod.GoodCop] = true, // Always available
            [InterrogationMethod.BadCop] = true,  // Always available
            [InterrogationMethod.Deception] = true, // Always available
            [InterrogationMethod.Bribery] = availableGold >= CalculateBriberyCost(state.ResistanceLevel),
            [InterrogationMethod.Torture] = true  // Always available (but with warnings)
        };

        return result;
    }

    #region Private Helper Methods

    private int CalculateInitialResistance(SubjectResistance level, int will, int modifiers)
    {
        // Base resistance from WILL
        var baseResistance = will;

        // Adjust based on resistance level
        baseResistance = level switch
        {
            SubjectResistance.Minimal => Math.Max(1, will - 1),
            SubjectResistance.Low => will,
            SubjectResistance.Moderate => will + 1,
            SubjectResistance.High => will + 2,
            SubjectResistance.Extreme => will + 4,
            _ => will
        };

        // Add modifiers
        baseResistance += modifiers;

        // Ensure within range
        var minChecks = level.GetMinChecksToBreak();
        var maxChecks = level.GetMaxChecksToBreak();

        return Math.Clamp(baseResistance, minChecks, maxChecks);
    }

    private int CalculateDc(InterrogationContext context)
    {
        var method = context.SelectedMethod;
        var baseDc = method.GetBaseDc();

        // Torture DC is special
        if (method == InterrogationMethod.Torture)
        {
            baseDc = CalculateTortureDc(
                context.InterrogationState.SubjectWill,
                context.DcModifier);
        }
        else
        {
            baseDc += context.DcModifier;
        }

        return baseDc;
    }

    private int CalculateDicePool(InterrogationContext context)
    {
        var method = context.SelectedMethod;
        int basePool;

        if (method == InterrogationMethod.Torture)
        {
            // Torture uses raw attribute
            basePool = context.UseMight
                ? context.InterrogatorAttribute
                : context.InterrogatorAttribute;
        }
        else
        {
            // Other methods use Attribute + Rhetoric
            basePool = context.InterrogatorAttribute + context.InterrogatorRhetoric;
        }

        // Add bonus dice
        basePool += context.BonusDice;

        return Math.Max(1, basePool);
    }

    private SkillOutcome DetermineOutcome(DiceRollResult roll, int successesRequired)
    {
        if (roll.IsFumble)
        {
            return SkillOutcome.CriticalFailure;
        }

        var excess = roll.NetSuccesses - successesRequired;

        return excess switch
        {
            >= 3 => SkillOutcome.CriticalSuccess,
            >= 0 => SkillOutcome.Success,
            _ => SkillOutcome.Failure
        };
    }

    private FumbleType GetFumbleTypeForMethod(InterrogationMethod method)
    {
        return method switch
        {
            InterrogationMethod.GoodCop => FumbleType.TrustShattered,
            InterrogationMethod.BadCop => FumbleType.ChallengeAccepted,
            InterrogationMethod.Deception => FumbleType.LieExposed,
            InterrogationMethod.Bribery => FumbleType.TrustShattered, // Insulting offer
            InterrogationMethod.Torture => FumbleType.SubjectBroken, // Subject dies/goes mad
            _ => FumbleType.TrustShattered
        };
    }

    private string GenerateRoundNarrative(
        InterrogationContext context,
        bool isSuccess,
        bool isFumble,
        InterrogationMethod method)
    {
        if (isFumble)
        {
            return method switch
            {
                InterrogationMethod.GoodCop => "Your attempt at empathy backfires completely. The subject sees through your facade and clams up.",
                InterrogationMethod.BadCop => "Your threats are met with defiance. The subject challenges you, emboldened rather than afraid.",
                InterrogationMethod.Deception => "Your lie is immediately spotted. The subject now knows you cannot be trusted.",
                InterrogationMethod.Bribery => "Your offer is taken as an insult. The subject's resolve hardens.",
                InterrogationMethod.Torture => "The torture goes too far. The subject is broken in ways that cannot be undone.",
                _ => "Something goes terribly wrong."
            };
        }

        if (isSuccess)
        {
            return method switch
            {
                InterrogationMethod.GoodCop => "Your sympathetic approach finds purchase. The subject seems slightly more willing to talk.",
                InterrogationMethod.BadCop => "Your aggressive stance has the desired effect. Fear flickers in the subject's eyes.",
                InterrogationMethod.Deception => "Your deception works. The subject unknowingly reveals something useful.",
                InterrogationMethod.Bribery => "The offer is tempting. The subject's resistance wavers.",
                InterrogationMethod.Torture => "The subject screams. Their will erodes under the pain.",
                _ => "The method has some effect."
            };
        }

        return method switch
        {
            InterrogationMethod.GoodCop => "The subject remains unmoved by your words.",
            InterrogationMethod.BadCop => "The subject endures your threats in sullen silence.",
            InterrogationMethod.Deception => "The subject doesn't take the bait.",
            InterrogationMethod.Bribery => "The offer isn't enough to sway them.",
            InterrogationMethod.Torture => "The subject holds on, refusing to break.",
            _ => "No progress is made."
        };
    }

    private string GenerateInformationContent(string topic, bool isTrue)
    {
        // In actual implementation, this would pull from game data
        // For now, return a placeholder indicating the topic
        var truthPrefix = isTrue ? "" : "[FALSE] ";
        return $"{truthPrefix}Information about: {topic}";
    }

    private string GenerateInterrogationSummary(InterrogationState state)
    {
        if (state.Status == InterrogationStatus.SubjectBroken)
        {
            var methodSummary = state.TortureUsed
                ? "through brutal methods that will not be forgotten"
                : $"primarily using {state.GetPrimaryMethod().GetDisplayName().ToLower()} techniques";

            return $"After {state.RoundNumber} rounds of interrogation {methodSummary}, " +
                   $"the subject's resistance finally crumbled. " +
                   $"Information reliability: {state.CalculateReliability()}%.";
        }

        if (state.Status == InterrogationStatus.Abandoned)
        {
            return $"The interrogation was abandoned after {state.RoundNumber} rounds. " +
                   "No reliable information was obtained.";
        }

        return $"The subject successfully resisted all attempts over {state.RoundNumber} rounds.";
    }

    #endregion
}
```

```csharp
// Location: src/Domain/Entities/FumbleConsequences/SubjectBrokenConsequence.cs

namespace RuneRust.Domain.Entities.FumbleConsequences;

using RuneRust.Domain.Enums;

/// <summary>
/// Represents the [Subject Broken] fumble consequence from torture.
/// The subject is damaged beyond recovery - they may die, go insane,
/// or become catatonic. No further information can be extracted.
/// </summary>
public sealed class SubjectBrokenConsequence : FumbleConsequence
{
    public override FumbleType Type => FumbleType.SubjectBroken;

    public override string Name => "Subject Broken";

    public override string Description =>
        "The torture has gone too far. The subject is broken beyond recovery - " +
        "their mind shattered or their body gave out. No reliable information " +
        "can ever be extracted from them again.";

    /// <summary>
    /// What happened to the subject.
    /// </summary>
    public SubjectBrokenOutcome Outcome { get; init; }

    /// <summary>
    /// Additional reputation loss beyond the standard -30.
    /// </summary>
    public int AdditionalReputationLoss { get; init; } = -20; // Total -50

    /// <summary>
    /// Whether witnesses will spread word of this atrocity.
    /// </summary>
    public bool WitnessesPresent { get; init; }

    /// <summary>
    /// Factions that will become hostile upon learning of this.
    /// </summary>
    public IReadOnlyList<string> HostileFactions { get; init; } = Array.Empty<string>();

    public override void Apply(GameState gameState, string targetId)
    {
        // Mark subject as broken/dead
        var subject = gameState.GetNpc(targetId);
        if (subject != null)
        {
            subject.SetCondition(Outcome switch
            {
                SubjectBrokenOutcome.Dead => NpcCondition.Dead,
                SubjectBrokenOutcome.Insane => NpcCondition.Insane,
                SubjectBrokenOutcome.Catatonic => NpcCondition.Catatonic,
                _ => NpcCondition.Traumatized
            });
        }

        // Apply reputation loss
        gameState.ModifyReputation(AdditionalReputationLoss);

        // Make factions hostile
        foreach (var factionId in HostileFactions)
        {
            gameState.SetFactionRelation(factionId, FactionRelation.Hostile);
        }

        // Potentially affect companions
```

**Actor:** Player Character
**Preconditions:** NPC is captured and available for interrogation
**Flow:**

4. System: DC 14 check, rolls 7d10, gets 4 successes → Success
5. Resistance: 5 → 4
6. Player continues with GoodCop for 2 more rounds (mixed results)
7. Player switches to BadCop to speed up
8. System warns: -5 disposition per round
9. Player confirms, BadCop succeeds
10. Resistance: 3 → 2
11. Two more rounds break the subject
12. Status: SubjectBroken
13. Player asks about hideout location
14. System extracts info with 85% reliability (mixed methods)

**Postconditions:** Information obtained, relationship damaged by -15

4. System calculates cost: 35 gold for Low resistance
5. System: DC 10 check, easy success
6. Resistance: 3 → 2, 35 gold spent
7. Player continues Bribery for 2 more rounds

9. Total cost: 105 gold

**Postconditions:** Information obtained, no relationship damage, gold spent
