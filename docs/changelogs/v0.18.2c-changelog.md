## Summary

Defined the `ICpsService` interface — the Application Layer contract for managing Cognitive Paradox Syndrome (CPS). This interface provides 8 methods organized into 4 functional regions.

---

## Added

### Interface

- **`ICpsService`** — CPS service contract with comprehensive XML documentation

### Methods

| Region           | Method                                | Returns                | Purpose                  |
| ---------------- | ------------------------------------- | ---------------------- | ------------------------ |
| State Query      | `GetCpsState(Guid)`                   | `CpsState`             | Full CPS snapshot        |
| State Query      | `GetCurrentStage(Guid)`               | `CpsStage`             | Current stage only       |
| State Query      | `GetUiEffects(Guid)`                  | `CpsUiEffects`         | UI distortion parameters |
| Stage Transition | `CheckStageChange(Guid, int, int)`    | `CpsStageChangeResult` | Detect stage transitions |
| Panic Table      | `RollPanicTable(Guid)`                | `PanicResult`          | d10 Panic Table roll     |
| Panic Table      | `ApplyPanicEffect(Guid, PanicResult)` | `void`                 | Apply panic effects      |
| Recovery         | `IsRecoverable(Guid)`                 | `bool`                 | Check if recoverable     |
| Recovery         | `GetRecoveryProtocol(Guid)`           | `CpsRecoveryProtocol`  | Recovery steps           |
