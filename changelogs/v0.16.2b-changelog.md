# v0.16.2b Changelog - Armor Category System

## Summary

Implements the **Armor Category System** that classifies armor into 5 categories (Light, Medium, Heavy, Shields, Specialized) with associated base penalties. This establishes category definitions and penalty data used by the Armor Proficiency System (v0.16.2).
