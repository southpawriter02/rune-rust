## Summary

Implements a persistent mini-map showing explored rooms, connections, and the player's current position. Unexplored rooms adjacent to visited areas show as "?" until discovered.

## Key Features

| Feature | Description |
|---------|-------------|
| **Exploration Tracking** | Session-based visited room tracking |
| **Adjacent Discovery** | Known but unvisited rooms shown as `?` |
| **Room Symbols** | `@`=current, `█`=explored, `?`=unknown |
| **Grid Rendering** | 5x3 room cells with connections |
| **Scroll Support** | `Scroll()` and `ResetScroll()` methods |
| **Event Notifications** | `OnRoomExplored` event for UI updates |

---

## Map Symbols

| Symbol | Meaning | Color |
|--------|---------|-------|
| `@` | Current room | Yellow |
| `█` | Explored room | White |
| `?` | Unexplored adjacent | DarkGray |
| `┌┐└┘─│` | Room borders | Gray |
