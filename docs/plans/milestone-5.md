# Milestone 5: The Saga (Content Expansion)

This phase focuses on long-term character growth, unlocking complex class mechanics, and deepening world interaction.

v0.4.0: Saga Progression
	•	v0.4.0a: The Legend (XP Engine) — Implement SagaService to track Legend accumulation from combat/exploration and trigger Milestone events.
	•	v0.4.0b: The Milestone (Rewards) — Implement level-up logic, granting Progression Points (PP) and stat bumps upon reaching thresholds.
	•	v0.4.0c: The Saga Menu (UI) — Create the TUI interface for spending PP on Attribute increases and Skill ranks.
v0.4.1: Specializations
	•	v0.4.1a: The Awakening (Unlock System) — Implement logic to unlock Specializations based on Archetype and prerequisites (quests/items).
	•	v0.4.1b: The Tree (Ability Structure) — Implement the tiered ability tree data structure (Tier 1 → Tier 3 → Capstone).
	•	v0.4.1c: The Trainer (UI) — Create the UI for viewing specialization trees and purchasing abilities with PP.
v0.4.2: Factions & Dialogue
	•	v0.4.2a: The Reputation (Factions) — Implement FactionService to track standing (e.g., Iron-Banes, Dvergr) and apply reaction modifiers.
	•	v0.4.2b: The Conversation (Dialogue Engine) — Implement a node-based dialogue system allowing branching conversations with NPCs.
	•	v0.4.2c: The Negotiator (Skill Checks) — Integrate Rhetoric/Intimidation checks into dialogue options to unlock secret paths or discounts.
