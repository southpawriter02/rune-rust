# Panic Table (CPS Stage 3)

*Ref: CPS Specification v5.0*

| d10 | Effect | Description | Mechanical Impact |
| --- | --- | --- | --- |
| :---: | :--- | :--- | :--- |
| **1-2** | **Freeze (Logic Lock)** | Character loses all actions this turn. Stares at a geometric anomaly. | **Stunned** (1 turn). Defense -5. |
| **3-4** | **Flight (Evacuation Protocol)** | Character spends all movement fleeing the source of stress. "Abort! Abort!" | Must use **Move** action away from source. |
| **5-6** | **Fight (Paradox Fury)** | Character attacks nearest entity (friend or foe). "Silence the noise!" | Must use **Attack** action on nearest target. |
| **7** | **Babble (Word Salad)** | Character shouts nonsensical technical jargon. | **Stealth** impossible for party. |
| **8** | **Self-Harm (Purge)** | Character inflicts damage to self to "let the pressure out." | Take **1d6** damage (ignores Soak). |
| **9** | **Hallucination (Phantom Data)** | Character treats empty space as cover or a wall. | **Disadvantage** on next Action. |
| **10** | **Catatonia (System Crash)** | Character falls Prone and unresponsive. | **Prone** and **Stunned** (1d4 turns). |
