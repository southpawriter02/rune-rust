# v0.13.4a Changelog: Achievement Panel

**Release Date:** 2026-01-21
**Version:** 0.13.4a
**Focus:** Achievement Panel UI

---

## Summary

Implements the Achievement Panel UI component for displaying achievements organized by category with progress bars, rarity badges, status indicators, and unlock notifications.

---

## Added

### Configuration
- `AchievementPanelConfig.cs` - Panel display configuration
- `config/achievement-panel.json` - Default configuration values
- `config/schemas/achievement-panel.schema.json` - JSON schema

### DTOs
- `AchievementDisplayDto.cs` - Combined achievement and progress data
- `AchievementProgressDisplayDto.cs` - Progress-specific display data  
- `AchievementCategoryDisplayDto.cs` - Category summary data

### Renderers
- `ProgressBarRenderer.cs` - ASCII progress bar rendering with color coding

### UI Components
- `RarityBadge.cs` - Tier badge rendering (Bronze/Silver/Gold/Platinum)
- `AchievementCard.cs` - Individual achievement card with status indicators
- `AchievementCategoryView.cs` - Category section with header and cards
- `AchievementPanel.cs` - Main panel with filtering and summary

### Notifications
- `AchievementUnlockNotification.cs` - Popup notification on unlock

### Commands
- `AchievementsCommand.cs` - Command handler for opening panel

### Unit Tests (44 new tests)
- `ProgressBarRendererTests.cs` - 10 tests
- `AchievementCardTests.cs` - 10 tests
- `AchievementPanelTests.cs` - 15 tests
- `AchievementUnlockNotificationTests.cs` - 9 tests

## Technical Notes

- All components follow established TUI patterns
