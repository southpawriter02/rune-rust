# Implementation Plan: v0.4.1b - The Unlock (Specialization Service Layer)

**Status:** Ready for Implementation
**Theme:** "The Acquisition"
**Scope:** Service layer for specialization unlocks, node purchases, prerequisite validation, and PP spending

---

## Executive Summary

v0.4.1b implements the **SpecializationService** - the business logic layer for the specialization system. Building on v0.4.1a's data foundation, this release enables:

- **Specialization Unlocks**: Characters can unlock new specialization paths (e.g., Berserkr, Skald)
- **Node Purchases**: Characters spend Progression Points (PP) to unlock ability nodes within trees
- **Prerequisite Validation**: Enforces parent node requirements, tier gates, and archetype restrictions
- **Ability Integration**: Unlocked nodes grant abilities to the character's active ability pool

**Key Patterns:**
- Result Pattern for operation outcomes (`NodeUnlockResult`, `SpecializationUnlockResult`)
- Interface-based design (`ISpecializationService`)
- Comprehensive structured logging (Serilog)
- 80%+ test coverage requirement

## 1. Decision Tree

```
START: Implement v0.4.1b
│
├─ Q: What result types are needed for operations?
│  ├─ SpecializationUnlockResult: For UnlockSpecialization()
│  └─ NodeUnlockResult: For UnlockNode()
│     └─ Both follow the Result Pattern (Success bool, Message, Details)
│
├─ Q: How to handle the "first specialization is free" rule?
│  ├─ Option A: Check if Character.UnlockedSpecializationIds.Count == 0
│  └─ Option B: Add Character.HasFreeSpecializationSlot property ← CHOSEN
│     └─ Reason: More explicit, allows future expansion (e.g., unlock tokens)
│
├─ Q: What is the PP cost structure for node unlocks?
│  └─ Use SpecializationNode.CostPP directly (set per-node in seed data)
│     ├─ Tier 1: 1 PP (but granted free with spec unlock)
│     ├─ Tier 2: 2 PP
│     ├─ Tier 3: 3 PP
│     └─ Tier 4 (Capstone): 5 PP
│
├─ Q: How to validate parent node prerequisites?
│  └─ SpecializationNode.ParentNodeIds (List<Guid>) checked against
│     Character.SpecializationProgress via HasNode() helper
│     └─ All parents must be unlocked (AND logic, not OR)
│
├─ Q: Should unlocking a specialization auto-unlock Tier 1 nodes?
│  ├─ Option A: Yes, grant all Tier 1 nodes automatically
│  └─ Option B: No, require manual purchase ← CHOSEN (per v0.4.1a design)
│     └─ Reason: Tier 1 nodes cost 1 PP, allowing player choice
│     └─ Exception: First specialization grants 1 free Tier 1 node (entry point)
│
├─ Q: Where does the unlocked ability get registered?
│  └─ SpecializationService calls CharacterSpecializationProgress.Add()
│     └─ AbilityService or CombatService reads from this to build ability list
│     └─ v0.4.1c (UI) will handle displaying unlocked abilities
│
├─ Q: How to handle archetype mismatch attempts?
│  └─ Return SpecializationUnlockResult.Failure with clear message
│     └─ "This specialization requires the Warrior archetype."
│
├─ Q: Should service layer persist changes or caller?
│  └─ Service calls SaveChangesAsync() internally after successful operations
│     └─ Follows repository transaction pattern from ProgressionService
│
└─ Q: How to integrate with existing Character entity?
   └─ Use existing Character methods:
      ├─ HasSpecialization(Guid specId) - Check if spec is unlocked
      ├─ HasNode(Guid nodeId) - Check if node is unlocked
      └─ ProgressionPoints property - PP balance
```

```csharp
namespace RuneAndRust.Core.Models.Specialization;

/// <summary>
/// Result of a specialization unlock attempt.
/// Follows the Result Pattern for explicit success/failure handling.
/// </summary>
public record SpecializationUnlockResult
{
    /// <summary>Whether the unlock operation succeeded.</summary>
    public bool IsSuccess { get; init; }

    /// <summary>Human-readable message describing the outcome.</summary>
    public string Message { get; init; } = string.Empty;

    /// <summary>The specialization ID that was unlocked (if successful).</summary>
    public Guid? SpecializationId { get; init; }

    /// <summary>The specialization name (for UI display).</summary>
    public string? SpecializationName { get; init; }

    /// <summary>PP cost that was deducted (0 for first spec).</summary>
    public int PpCost { get; init; }

    /// <summary>Character's remaining PP after the operation.</summary>
    public int PpRemaining { get; init; }

    /// <summary>
    /// Creates a successful unlock result.
    /// </summary>
    public static SpecializationUnlockResult Success(
        Guid specId,
        string specName,
        int ppCost,
        int ppRemaining)
    {
        return new SpecializationUnlockResult
        {
            IsSuccess = true,
            Message = $"You have unlocked the {specName} specialization!",
            SpecializationId = specId,
            SpecializationName = specName,
            PpCost = ppCost,
            PpRemaining = ppRemaining
        };
    }

    /// <summary>
    /// Creates a failure result with explanation.
    /// </summary>
    public static SpecializationUnlockResult Failure(string reason)
    {
        return new SpecializationUnlockResult
        {
            IsSuccess = false,
            Message = reason,
            SpecializationId = null,
            SpecializationName = null,
            PpCost = 0,
            PpRemaining = 0
        };
    }
}
```

```csharp
namespace RuneAndRust.Core.Models.Specialization;

/// <summary>
/// Result of a specialization node unlock attempt.
/// Follows the Result Pattern for explicit success/failure handling.
/// </summary>
public record NodeUnlockResult
{
    /// <summary>Whether the unlock operation succeeded.</summary>
    public bool IsSuccess { get; init; }

    /// <summary>Human-readable message describing the outcome.</summary>
    public string Message { get; init; } = string.Empty;

    /// <summary>The node ID that was unlocked (if successful).</summary>
    public Guid? NodeId { get; init; }

    /// <summary>The ability name granted by this node.</summary>
    public string? AbilityName { get; init; }

    /// <summary>The tier of the unlocked node (1-4).</summary>
    public int Tier { get; init; }

    /// <summary>PP cost that was deducted.</summary>
    public int PpCost { get; init; }

    /// <summary>Character's remaining PP after the operation.</summary>
    public int PpRemaining { get; init; }

    /// <summary>Whether this was a capstone (Tier 4) ability.</summary>
    public bool IsCapstone { get; init; }

    /// <summary>
    /// Creates a successful node unlock result.
    /// </summary>
    public static NodeUnlockResult Success(
        Guid nodeId,
        string abilityName,
        int tier,
        int ppCost,
        int ppRemaining,
        bool isCapstone)
    {
        string message = isCapstone
            ? $"You have mastered the {abilityName} capstone ability!"
            : $"You have learned {abilityName}!";

        return new NodeUnlockResult
        {
            IsSuccess = true,
            Message = message,
            NodeId = nodeId,
            AbilityName = abilityName,
            Tier = tier,
            PpCost = ppCost,
            PpRemaining = ppRemaining,
            IsCapstone = isCapstone
        };
    }

    /// <summary>
    /// Creates a failure result with explanation.
    /// </summary>
    public static NodeUnlockResult Failure(string reason)
    {
        return new NodeUnlockResult
        {
            IsSuccess = false,
            Message = reason,
            NodeId = null,
            AbilityName = null,
            Tier = 0,
            PpCost = 0,
            PpRemaining = 0,
            IsCapstone = false
        };
    }

    /// <summary>
    /// Creates a failure result indicating missing prerequisites.
    /// </summary>
    public static NodeUnlockResult MissingPrerequisites(IEnumerable<string> missingNodeNames)
    {
        var names = string.Join(", ", missingNodeNames);
        return Failure($"Prerequisites not met. You must first unlock: {names}");
    }
}
```

```csharp
using RuneAndRust.Core.Entities;
using RuneAndRust.Core.Enums;
using RuneAndRust.Core.Models.Specialization;

namespace RuneAndRust.Core.Interfaces;

/// <summary>
/// Service for managing character specialization unlocks and node purchases.
/// Handles business logic for the specialization tree system.
/// </summary>
/// <remarks>See: v0.4.1b (The Unlock) for system design.</remarks>
public interface ISpecializationService
{
    #region Specialization Queries

    /// <summary>
    /// Gets all specializations available to a character based on their archetype.
    /// Includes both locked and unlocked specializations.
    /// </summary>
    /// <param name="character">The character to query for.</param>
    /// <returns>All specializations matching the character's archetype.</returns>
    Task<IEnumerable<Specialization>> GetAvailableSpecializationsAsync(Character character);

    /// <summary>
    /// Checks if a character can unlock a specific specialization.
    /// Validates archetype, level, PP cost, and existing unlock status.
    /// </summary>
    /// <param name="character">The character attempting the unlock.</param>
    /// <param name="specializationId">The specialization to check.</param>
    /// <returns>True if all requirements are met.</returns>
    Task<bool> CanUnlockSpecializationAsync(Character character, Guid specializationId);

    /// <summary>
    /// Attempts to unlock a specialization for the character.
    /// Deducts PP (0 for first spec, 3 for subsequent) and updates character state.
    /// </summary>
    /// <param name="character">The character unlocking the specialization.</param>
    /// <param name="specializationId">The specialization to unlock.</param>
    /// <returns>Result indicating success or failure with details.</returns>
    Task<SpecializationUnlockResult> UnlockSpecializationAsync(Character character, Guid specializationId);

    #endregion

    #region Node Queries

    /// <summary>
    /// Gets all nodes the character can currently unlock within a specialization.
    /// Filters to nodes where all prerequisites are met and node is not yet owned.
    /// </summary>
    /// <param name="character">The character to check for.</param>
    /// <param name="specializationId">The specialization to query.</param>
    /// <returns>Nodes that can be purchased with sufficient PP.</returns>
    Task<IEnumerable<SpecializationNode>> GetUnlockableNodesAsync(Character character, Guid specializationId);

    /// <summary>
    /// Gets all nodes the character has already unlocked across all specializations.
    /// </summary>
    /// <param name="character">The character to query.</param>
    /// <returns>All unlocked nodes with their abilities loaded.</returns>
    Task<IEnumerable<SpecializationNode>> GetUnlockedNodesAsync(Character character);

    /// <summary>
    /// Checks if a character can unlock a specific node.
    /// Validates spec ownership, parent prerequisites, PP cost, and existing unlock status.
    /// </summary>
    /// <param name="character">The character attempting the unlock.</param>
    /// <param name="nodeId">The node to check.</param>
    /// <returns>True if all requirements are met.</returns>
    Task<bool> CanUnlockNodeAsync(Character character, Guid nodeId);

    /// <summary>
    /// Attempts to unlock a node for the character.
    /// Deducts PP and records the unlock in CharacterSpecializationProgress.
    /// </summary>
    /// <param name="character">The character unlocking the node.</param>
    /// <param name="nodeId">The node to unlock.</param>
    /// <returns>Result indicating success or failure with details.</returns>
    Task<NodeUnlockResult> UnlockNodeAsync(Character character, Guid nodeId);

    #endregion

    #region Utility

    /// <summary>
    /// Calculates the PP cost to unlock a specialization for a character.
    /// Returns 0 if this would be their first specialization, otherwise 3.
    /// </summary>
    /// <param name="character">The character to calculate for.</param>
    /// <returns>PP cost (0 or 3).</returns>
    int GetSpecializationUnlockCost(Character character);

    /// <summary>
    /// Gets the total PP invested in a specific specialization tree.
    /// Useful for UI display of tree progress.
    /// </summary>
    /// <param name="character">The character to query.</param>
    /// <param name="specializationId">The specialization to sum.</param>
    /// <returns>Total PP spent on nodes in this tree.</returns>
    Task<int> GetPpInvestedInTreeAsync(Character character, Guid specializationId);

    #endregion
}
```

## 7. Phase 3: Service Implementation

**File:** `RuneAndRust.Engine/Services/SpecializationService.cs`

```csharp
using Microsoft.Extensions.Logging;
using RuneAndRust.Core.Entities;
using RuneAndRust.Core.Interfaces;
using RuneAndRust.Core.Models.Specialization;

namespace RuneAndRust.Engine.Services;

/// <summary>
/// Engine-layer service for managing specialization unlocks and node purchases.
/// Handles PP spending, prerequisite validation, and state persistence.
/// </summary>
/// <remarks>See: v0.4.1b (The Unlock) for system design.</remarks>
public class SpecializationService : ISpecializationService
{
    private readonly ISpecializationRepository _repository;
    private readonly ICharacterRepository _characterRepository;
    private readonly ILogger<SpecializationService> _logger;

    /// <summary>
    /// PP cost to unlock a specialization (after the first free one).
    /// </summary>
    private const int SpecializationUnlockCost = 3;

    /// <summary>
    /// Number of free specialization slots each character gets.
    /// </summary>
    private const int FreeSpecializationCount = 1;

    public SpecializationService(
        ISpecializationRepository repository,
        ICharacterRepository characterRepository,
        ILogger<SpecializationService> logger)
    {
        _repository = repository;
        _characterRepository = characterRepository;
        _logger = logger;
    }

    #region Specialization Operations

    /// <inheritdoc/>
    public async Task<IEnumerable<Specialization>> GetAvailableSpecializationsAsync(Character character)
    {
        _logger.LogDebug(
            "[SpecService] GetAvailableSpecializationsAsync for {Name} (Archetype: {Archetype})",
            character.Name, character.Archetype);

        var specs = await _repository.GetByArchetypeAsync(character.Archetype);

        _logger.LogTrace(
            "[SpecService] Found {Count} specializations for {Archetype}",
            specs.Count(), character.Archetype);

        return specs;
    }

    /// <inheritdoc/>
    public async Task<bool> CanUnlockSpecializationAsync(Character character, Guid specializationId)
    {
        _logger.LogTrace(
            "[SpecService] CanUnlockSpecializationAsync: {Name} -> {SpecId}",
            character.Name, specializationId);

        var spec = await _repository.GetByIdAsync(specializationId);
        if (spec == null)
        {
            _logger.LogDebug("[SpecService] Specialization {SpecId} not found", specializationId);
            return false;
        }

        var (isValid, _) = ValidateSpecializationUnlock(character, spec);
        return isValid;
    }

    /// <inheritdoc/>
    public async Task<SpecializationUnlockResult> UnlockSpecializationAsync(
        Character character,
        Guid specializationId)
    {
        _logger.LogDebug(
            "[SpecService] UnlockSpecializationAsync: {Name} attempting to unlock {SpecId}",
            character.Name, specializationId);

        // 1. Fetch specialization
        var spec = await _repository.GetByIdAsync(specializationId);
        if (spec == null)
        {
            _logger.LogWarning(
                "[SpecService] Unlock failed: Specialization {SpecId} not found",
                specializationId);
            return SpecializationUnlockResult.Failure("Specialization not found.");
        }

        // 2. Validate all requirements
        var (isValid, failureReason) = ValidateSpecializationUnlock(character, spec);
        if (!isValid)
        {
            _logger.LogWarning(
                "[SpecService] Unlock failed for {Name}: {Reason}",
                character.Name, failureReason);
            return SpecializationUnlockResult.Failure(failureReason);
        }

        // 3. Calculate cost and execute transaction
        int cost = GetSpecializationUnlockCost(character);
        int ppBefore = character.ProgressionPoints;

        character.ProgressionPoints -= cost;
        character.UnlockedSpecializationIds.Add(spec.Id);

        // 4. Persist changes
        await _characterRepository.UpdateAsync(character);
        await _characterRepository.SaveAsync();

        _logger.LogInformation(
            "[SpecService] {Name} unlocked {SpecName} (Cost: {Cost} PP, Remaining: {Remaining} PP)",
            character.Name, spec.Name, cost, character.ProgressionPoints);

        return SpecializationUnlockResult.Success(
            spec.Id,
            spec.Name,
            cost,
            character.ProgressionPoints);
    }

    /// <inheritdoc/>
    public int GetSpecializationUnlockCost(Character character)
    {
        bool isFree = character.UnlockedSpecializationIds.Count < FreeSpecializationCount;
        return isFree ? 0 : SpecializationUnlockCost;
    }

    /// <summary>
    /// Validates all requirements for unlocking a specialization.
    /// </summary>
    private (bool IsValid, string FailureReason) ValidateSpecializationUnlock(
        Character character,
        Specialization spec)
    {
        // Check archetype match
        if (spec.RequiredArchetype != character.Archetype)
        {
            return (false, $"This specialization requires the {spec.RequiredArchetype} archetype.");
        }

        // Check level requirement
        if (character.Level < spec.RequiredLevel)
        {
            return (false, $"You must be level {spec.RequiredLevel} to unlock this specialization (current: {character.Level}).");
        }

        // Check not already unlocked
        if (character.HasSpecialization(spec.Id))
        {
            return (false, "You have already unlocked this specialization.");
        }

        // Check PP cost
        int cost = GetSpecializationUnlockCost(character);
        if (character.ProgressionPoints < cost)
        {
            return (false, $"Insufficient Progression Points. Need {cost}, have {character.ProgressionPoints}.");
        }

        return (true, string.Empty);
    }

    #endregion

    #region Node Operations

    /// <inheritdoc/>
    public async Task<IEnumerable<SpecializationNode>> GetUnlockableNodesAsync(
        Character character,
        Guid specializationId)
    {
        _logger.LogDebug(
            "[SpecService] GetUnlockableNodesAsync: {Name} for spec {SpecId}",
            character.Name, specializationId);

        // Must own the specialization
        if (!character.HasSpecialization(specializationId))
        {
            _logger.LogTrace("[SpecService] Character does not own spec {SpecId}", specializationId);
            return Enumerable.Empty<SpecializationNode>();
        }

        var allNodes = await _repository.GetNodesForSpecializationAsync(specializationId);
        var unlockable = new List<SpecializationNode>();

        foreach (var node in allNodes)
        {
            // Skip already owned
            if (character.HasNode(node.Id))
                continue;

            // Check prerequisites
            if (!AreAllParentsUnlocked(character, node))
                continue;

            unlockable.Add(node);
        }

        _logger.LogTrace(
            "[SpecService] Found {Count} unlockable nodes for {Name} in spec {SpecId}",
            unlockable.Count, character.Name, specializationId);

        return unlockable;
    }

    /// <inheritdoc/>
    public async Task<IEnumerable<SpecializationNode>> GetUnlockedNodesAsync(Character character)
    {
        _logger.LogDebug("[SpecService] GetUnlockedNodesAsync for {Name}", character.Name);

        var nodes = await _repository.GetUnlockedNodesAsync(character.Id);

        _logger.LogTrace(
            "[SpecService] Character {Name} has {Count} unlocked nodes",
            character.Name, nodes.Count());

        return nodes;
    }

    /// <inheritdoc/>
    public async Task<bool> CanUnlockNodeAsync(Character character, Guid nodeId)
    {
        _logger.LogTrace(
            "[SpecService] CanUnlockNodeAsync: {Name} -> {NodeId}",
            character.Name, nodeId);

        var node = await _repository.GetNodeByIdAsync(nodeId);
        if (node == null)
        {
            _logger.LogDebug("[SpecService] Node {NodeId} not found", nodeId);
            return false;
        }

        var (isValid, _) = ValidateNodeUnlock(character, node);
        return isValid;
    }

    /// <inheritdoc/>
    public async Task<NodeUnlockResult> UnlockNodeAsync(Character character, Guid nodeId)
    {
        _logger.LogDebug(
            "[SpecService] UnlockNodeAsync: {Name} attempting to unlock node {NodeId}",
            character.Name, nodeId);

        // 1. Fetch node with ability
        var node = await _repository.GetNodeByIdAsync(nodeId);
        if (node == null)
        {
            _logger.LogWarning("[SpecService] Unlock failed: Node {NodeId} not found", nodeId);
            return NodeUnlockResult.Failure("Node not found.");
        }

        // 2. Validate all requirements
        var (isValid, failureReason) = ValidateNodeUnlock(character, node);
        if (!isValid)
        {
            // Check for missing prerequisites specifically
            if (failureReason.Contains("Prerequisites"))
            {
                var missingParents = await GetMissingParentNamesAsync(character, node);
                _logger.LogWarning(
                    "[SpecService] Node unlock failed for {Name}: Missing prereqs {Parents}",
                    character.Name, string.Join(", ", missingParents));
                return NodeUnlockResult.MissingPrerequisites(missingParents);
            }

            _logger.LogWarning(
                "[SpecService] Node unlock failed for {Name}: {Reason}",
                character.Name, failureReason);
            return NodeUnlockResult.Failure(failureReason);
        }

        // 3. Execute transaction
        int cost = node.CostPP;
        character.ProgressionPoints -= cost;

        await _repository.RecordNodeUnlockAsync(character.Id, node.Id);
        await _characterRepository.UpdateAsync(character);
        await _repository.SaveChangesAsync();

        string abilityName = node.GetDisplayName();
        _logger.LogInformation(
            "[SpecService] {Name} unlocked node {AbilityName} (Tier {Tier}, Cost: {Cost} PP, Remaining: {Remaining} PP){Capstone}",
            character.Name, abilityName, node.Tier, cost, character.ProgressionPoints,
            node.IsCapstone ? " [CAPSTONE]" : "");

        return NodeUnlockResult.Success(
            node.Id,
            abilityName,
            node.Tier,
            cost,
            character.ProgressionPoints,
            node.IsCapstone);
    }

    /// <inheritdoc/>
    public async Task<int> GetPpInvestedInTreeAsync(Character character, Guid specializationId)
    {
        _logger.LogTrace(
            "[SpecService] GetPpInvestedInTreeAsync: {Name} for spec {SpecId}",
            character.Name, specializationId);

        var unlockedNodes = await _repository.GetUnlockedNodesAsync(character.Id);
        int total = unlockedNodes
            .Where(n => n.SpecializationId == specializationId)
            .Sum(n => n.CostPP);

        _logger.LogTrace("[SpecService] Total PP invested: {Total}", total);
        return total;
    }

    /// <summary>
    /// Validates all requirements for unlocking a node.
    /// </summary>
    private (bool IsValid, string FailureReason) ValidateNodeUnlock(
        Character character,
        SpecializationNode node)
    {
        // Check specialization is owned
        if (!character.HasSpecialization(node.SpecializationId))
        {
            return (false, "You must first unlock this specialization.");
        }

        // Check not already owned
        if (character.HasNode(node.Id))
        {
            return (false, "You have already unlocked this ability.");
        }

        // Check prerequisites
        if (!AreAllParentsUnlocked(character, node))
        {
            return (false, "Prerequisites not met.");
        }

        // Check PP cost
        if (character.ProgressionPoints < node.CostPP)
        {
            return (false, $"Insufficient Progression Points. Need {node.CostPP}, have {character.ProgressionPoints}.");
        }

        return (true, string.Empty);
    }

    /// <summary>
    /// Checks if all parent nodes are unlocked for the given node.
    /// </summary>
    private bool AreAllParentsUnlocked(Character character, SpecializationNode node)
    {
        // Tier 1 nodes have no prerequisites
        if (node.ParentNodeIds.Count == 0)
            return true;

        return node.ParentNodeIds.All(parentId => character.HasNode(parentId));
    }

    /// <summary>
    /// Gets the names of missing parent nodes for error messages.
    /// </summary>
    private async Task<IEnumerable<string>> GetMissingParentNamesAsync(
        Character character,
        SpecializationNode node)
    {
        var missing = new List<string>();

        foreach (var parentId in node.ParentNodeIds)
        {
            if (!character.HasNode(parentId))
            {
                var parent = await _repository.GetNodeByIdAsync(parentId);
                if (parent != null)
                {
                    missing.Add(parent.GetDisplayName());
                }
            }
        }

        return missing;
    }

    #endregion
}
```
