# v0.20.2b Changelog — Rúnasmiðr Tier 2 Abilities

## Added

### Empowered Inscription (Active Ability)
- `EmpoweredRune` value object with elemental type validation (fire, cold, lightning, aetheric)
- +1d6 elemental damage bonus to weapons for 10 turns
- Cost: 4 AP + 2 Rune Charges
- Stacks with base Inscribe Rune; replaces existing empowered rune on same weapon
- Prerequisite: 8+ PP invested in Rúnasmiðr tree

### Runic Trap (Active Ability)
- `RunicTrap` value object with trigger/expiry lifecycle
- `TrapTriggerType` enum and `TrapTriggerResult` sealed record
- 3d6 damage, DC 14 Perception to detect
- Maximum 3 active traps per player, 1-hour expiration
- Cost: 3 AP + 2 Rune Charges
- Prerequisite: 8+ PP invested in Rúnasmiðr tree

### Dvergr Techniques (Passive Ability)
- `ICraftingCostModifier` interface
- `DvergrTechniquesService` implementing 20% crafting cost/time reduction
- Does NOT affect Rune Charge or special resource costs
- Prerequisite: 8+ PP invested in Rúnasmiðr tree

### Player Entity Extensions
- `EmpoweredRunes` collection property with Add/Remove methods
- `ActiveTraps` collection property with Add/Remove/GetActive methods
- `HasDvergrTechniques` computed property

### Configuration
- Added `empowered-inscription`, `runic-trap`, and `dvergr-techniques` to `abilities.json`

## Changed

### IRunasmidrAbilityService
- Added `ExecuteEmpoweredInscription`, `ExecuteRunicTrap`, `GetEmpoweredRunes`, `GetActiveTraps`
- Updated documentation to include Tier 2 ability list

### RunasmidrAbilityService
- Added Tier 2 constants (AP/Charge costs)
- Implemented full validation and execution for Empowered Inscription and Runic Trap
- Added query methods for empowered runes and active traps
