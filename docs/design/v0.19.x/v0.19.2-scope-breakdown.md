# v0.19.2 Scope Breakdown: Svartalfheim & Helheim

## Overview

v0.19.2 implements two interconnected subterranean realms:
- **Svartalfheim — The Dvergr Forges** (Deck 06) — Manufacturing realm with light-stratified zones
- **Helheim — The Gangrenous Gut** (Deck 08) — Toxic catastrophe biome and source of world-wide contamination

These realms share a critical connection via Garm's Watch, the sealed elevator shaft preventing Helheim's contamination from ascending.

## Application Layer Specifications

### ISvartalfheimService Interface

```csharp
/// <summary>
/// Manages Svartalfheim-specific mechanics.
/// </summary>
public interface ISvartalfheimService
{
    /// <summary>Gets the darkness level at a location.</summary>
    DarknessLevel GetDarknessLevel(int x, int y, int z, bool hasLightSource);

    /// <summary>Checks if a light source attracts attention in Black Veins.</summary>
    bool CheckLightAttraction(int x, int y, int z, int lightRadius);

    /// <summary>Gets available Anvil-Star coverage at a location.</summary>
    bool HasAnvilStarCoverage(int x, int y, int z);

    /// <summary>Calculates perception penalty from darkness.</summary>
    int CalculateDarknessPerceptionPenalty(DarknessLevel darkness);
}
```

### IHelheimService Interface

```csharp
/// <summary>
/// Manages Helheim-specific mechanics.
/// </summary>
public interface IHelheimService
{
    /// <summary>Gets the toxic exposure level at a location.</summary>
    ToxicExposureLevel GetToxicExposure(
        HelheimStratumId stratum,
        bool hasEnvironmentalSeal,
        bool hasFiltration);

    /// <summary>Gets structural integrity at a location.</summary>
    StructuralIntegrityState GetStructuralIntegrity(int x, int y, int z);

    /// <summary>Performs a step-test to check structural safety.</summary>
    StructuralTestResult PerformStepTest(int x, int y, int z, int skill);

    /// <summary>Checks for Hvergelmir proximity CPS intensity.</summary>
    int GetHvergelmirCpsIntensity(int distanceFromAnomaly);

    /// <summary>Gets available hazards at a location.</summary>
    IReadOnlyList<HelheimHazard> GetHazardsAtLocation(
        HelheimStratumId stratum,
        int x, int y, int z);

    /// <summary>Calculates corrosion damage to equipment.</summary>
    int CalculateCorrosionDamage(int exposureHours, float corrosionRate);
}
```
