# v0.18.3d Changelog — ITraumaService Interface

**Release Date:** 2026-02-01  
**Version:** 0.18.3d

## Summary

Defines the `ITraumaService` interface — the central service contract for all trauma management operations.

---

## Added

### Application Layer — Interfaces

- `ITraumaService.cs` — 9-method service contract for trauma operations
    - **Query Methods:** `GetTraumasAsync`, `GetTraumaDefinitionAsync`, `HasTraumaAsync`, `GetTraumaCountAsync`, `GetTraumaStackCountAsync`, `GetActiveEffectsAsync`
    - **Mutation Methods:** `AcquireTraumaAsync`, `CheckRetirementConditionAsync`
    - **Check Methods:** `PerformTraumaCheckAsync`
    - Comprehensive XML documentation with examples
