# v0.0.3 Changelog: Player Creation & Text Variety Systems

**Release Date:** 2025-01-06
**Version:** 0.0.3

## Overview

This release introduces the complete character creation system and text variety systems for dynamic, non-repetitive game narration.

---

## Stage A: Core Player Creation

### Character Creation Features
- **Name Validation**: 2-30 characters, letters/spaces/hyphens/apostrophes only
- **Race Selection**: 4 playable races with unique modifiers and traits
  - Human (balanced), Elf (agile), Dwarf (sturdy), Halfling (lucky)
- **Background Selection**: 8 backgrounds with bonuses and abilities
  - Professions: Soldier, Scholar, Thief, Healer
  - Lineages: Mountainborn, Coastlander, Forestkin, Cityfolk
- **Point-Buy Attributes**: 25 points across 5 attributes
  - Might, Fortitude, Will, Wits, Finesse
- **Character Description**: Optional 500-character backstory
- **Character Summary**: Review and confirm before starting

## Stage B: Text Variety Systems

### Lexicon System
- Context-aware synonym selection (combat, formal, casual, etc.)
- Weighted random selection with configurable probabilities
- Severity-based terms for damage, quantities, conditions
- 8 term categories with 40+ synonyms each

### Descriptor System
- 9 descriptor category files with 30+ pools
- Theme-based filtering and emphasis
- Anti-repetition tracking (recent history avoidance)
- Damage percentage-based descriptor selection
- Combined description generation from multiple pools

### Theme System
- Dark Fantasy, High Fantasy, Horror presets
- Descriptor overrides per theme
- Excluded and emphasized term lists
- Runtime theme switching support
