# v0.8.0a Changelog - Character Creation Wizard

## Added

### ViewModels
- `WizardViewModel` - Main wizard controller with step navigation
- `WizardStepViewModelBase` - Abstract base class for wizard steps
- `NameEntryStepViewModel` - Character name input with validation
- `RaceSelectionStepViewModel` - Race selection with sample races (Human, Elf, Dwarf)
- `ClassSelectionStepViewModel` - Class selection with sample classes (Warrior, Mage, Rogue)
- `StatAllocationStepViewModel` - Point-buy stat allocation system
- `AppearanceStepViewModel` - Portrait selection
- `ReviewStepViewModel` - Character summary review
- `StatAllocationViewModel` - Individual stat with increment/decrement
- `StepIndicator` - Progress indicator dot

### Views
- `CharacterCreationWizard.axaml` - Wizard window with all step DataTemplates

### Models
- `CharacterCreationData` - Wizard state container
- `PortraitOption` - Portrait selection option

### Styles
- `WizardStyles.axaml` - Wizard-specific styles

### Tests
- 12 new unit tests covering name validation, race/class selection, stat allocation, and navigation

## Technical Notes
- Uses `CommunityToolkit.Mvvm` for `ObservableProperty` and `RelayCommand`
- Point-buy system: 27 points with costs 0-9 for stats 8-15
- DataTemplates defined directly in wizard window for simplicity
- Step ViewModels include sample data for demo purposes
