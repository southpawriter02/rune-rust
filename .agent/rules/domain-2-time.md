## Domain 2: Timeline Constraints

### Canonical Ground Truth

**5-Age Chronology:**

1.  **Age of Forging** (Cycle notation: 1-999)
2.  **The Ginnungagap Glitch** (Year 0 — zero-point event)
3.  **Age of Echoes** (PG notation: 1-783 PG current)
4.  **The Age Before** (PRE-GLITCH umbrella term)
5.  **Future Ages** (not yet defined)

**Timeline Rules:**

*   Ginnungagap Glitch = Year 0 (absolute reference point)
*   All current events use PG notation (Post-Glitch)
*   Pre-Glitch uses Cycle notation (Age of Forging)
*   Current year: **783 PG**

### Audit Checklist

*   \[ \] All contemporary dates use PG notation?
*   \[ \] Pre-Glitch dates use Cycle notation?
*   \[ \] No absolute calendar dates (e.g., "2347 AD")?
*   \[ \] Event sequences align with 5-age chronology?
*   \[ \] Timeline compression avoided (Ages span appropriate durations)?
*   \[ \] Ginnungagap Glitch positioned as Year 0 pivot?

### Common Violations

❌ Absolute calendar dates (AD, CE, etc.)
❌ Mixed notation (PG + Cycle in same timeframe)
❌ Timeline contradictions (events out of sequence)
❌ Undefined age references

### Green Flags

✅ Consistent PG notation for current era
✅ Cycle notation for Age of Forging
✅ Year 0 = Ginnungagap Glitch
✅ 783 PG as current year
