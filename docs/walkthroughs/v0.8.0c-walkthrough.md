# v0.8.0c Walkthrough - Keyboard Shortcuts

## Overview

Implemented a keyboard shortcut service with context-aware processing. Global shortcuts (F1, F5, F9, Escape) work everywhere. Game and Combat shortcuts only work in their respective contexts.

## Architecture

```
ProcessKeyDown(key, modifiers)
    ├── Check Global bindings FIRST
    │   └── Execute if match found
    └── Check CurrentContext bindings
        └── Execute if match found
```

## Key Features

- **Context Priority**: Global > Current Context
- **Overlay (F1)**: Shows all shortcuts grouped by context
- **Conflict Detection**: Prevents duplicate bindings in same context
- **Reset to Defaults**: Restores all bindings
