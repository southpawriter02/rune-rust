## Added

### Domain Layer

- **`LivingRuneEntity`** — Immutable sealed record representing an animated rune entity summoned by the Living Runes ability
    - 10 HP, 12 DEF, +4 ATK, 1d8 DMG, 4 Movement, 5-turn duration
    - Factory method `Create(ownerId, position)` with all default constants
    - State transitions via `with` expressions: `TakeDamage`, `Tick`, `MoveTo`, `RollAttackDamage`
    - Status display and HP bar rendering
- **`DispelEffectResult`** — Immutable sealed record reporting Word of Unmaking dispel results
    - Tracks effects removed, entities destroyed, items affected, characters impacted
    - Factory methods `Create(...)` and `Empty()` with automatic total computation
    - Combat log summary display

### Application Layer

- **`RunasmidrTier3AbilityService`** — Service implementing Tier 3 and Capstone abilities
    - **Master Scrivener** — Passive 2x duration multiplier for runes and traps
    - **Living Runes** — Summons 2 animated rune entities with default stats
    - **Word of Unmaking** — Builds dispel results from area effect data; once-per-combat check
    - Tier 3 (16 PP) and Capstone (24 PP) prerequisite checks
    - Structured logging (no string interpolation) on all operations

### Configuration

- Updated `specializations.json` with Tier 3 ("Master Inscription") and Tier 4 ("Word of Unmaking") ability tiers for Rúnasmiðr
