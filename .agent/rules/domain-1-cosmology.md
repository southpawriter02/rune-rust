## Domain 1: Cosmology Constraints

### Canonical Ground Truth

**Nine Realms Structure:**

*   Yggdrasil = Deck-based megastructure (not literal tree)
*   9 Decks: Asgard (01), Vanaheim (03), Alfheim (04), Midgard (05), Svartalfheim (06), Jötunheim (07), Muspelheim (08), Helheim (09), Niflheim (10)
*   The Deep = Subterranean network beneath all decks

**Critical Constraint:**

*   Vanaheim/Alfheim are **laterally adjacent** to Midgard (not overhead)
*   No "upper/lower realm" terminology
*   Deck numbering ≠ vertical stacking order

### Audit Checklist

*   \[ \] Realm assignment matches canonical Nine Realms?
*   \[ \] No references to "upper realms" or "lower realms"?
*   \[ \] Vanaheim/Alfheim positioned correctly (lateral, not overhead)?
*   \[ \] Deck function aligns with Setting Fundamentals description?
*   \[ \] No contradictions with Yggdrasil megastructure cosmology?
*   \[ \] The Deep references appropriate (subterranean network context)?

### Common Violations

❌ "Upper realms" / "lower realms" terminology
❌ Vanaheim "above" Midgard or "canopy overhead"
❌ Contradictory deck functions
❌ Non-canonical realm additions

### Green Flags

✅ Deck-based architecture language
✅ Lateral realm relationships
✅ Subterranean Deep references
✅ Canonical Nine Realms only
