## Summary

Implements combat and ability sound service interfaces with damage type and ability school mappings.

---

## Added

### Application Layer - Interfaces

| Interface | Description |
|-----------|-------------|
| `ICombatSoundService` | Combat sound triggering |
| `IAbilitySoundService` | Ability sound triggering |

**ICombatSoundService Methods:**
- `PlayAttackHit(isCritical)` — Normal or critical hit
- `PlayAttackMiss()` — Miss sound
- `PlayAttackBlocked()` — Block sound
- `PlayDamage(type)` — Damage type sound
- `PlayMonsterDeath()` — Monster death
- `PlayPlayerDeath()` — Player death

**IAbilitySoundService Methods:**
- `PlayAbilityCast(school)` — School-specific cast
- `PlayBuffApplied()` — Buff sound
- `PlayDebuffApplied()` — Debuff sound
- `PlayHealReceived()` — Heal sound
- `PlayEffectExpired()` — Expiration sound

### Infrastructure Layer - Services

| Class | Description |
|-------|-------------|
| `CombatSoundService` | Combat sound implementation |
| `AbilitySoundService` | Ability sound implementation |

### Sound Mappings

**Damage Types:**
| Input | Effect ID |
|-------|-----------|
| fire | damage-fire |
| ice, cold, frost | damage-ice |
| lightning, electric, shock | damage-lightning |
| poison | damage-poison |
| holy, radiant | damage-holy |
| shadow, dark, necrotic | damage-shadow |

**Ability Schools:**
| Input | Effect ID |
|-------|-----------|
| fire, evocation, pyromancy | ability-fire |
| ice, cold, frost, cryomancy | ability-ice |
| lightning, storm, electromancy | ability-lightning |
| healing, restoration, holy | ability-heal |
| buff, enhancement, abjuration | ability-buff |
| shadow, dark, necromancy | ability-shadow |
| nature, druid | ability-nature |
| (unknown) | ability-cast |
