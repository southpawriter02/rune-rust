# v0.8.0b Changelog - Settings Window

## Added

### Models
- `GameSettings` - Aggregate settings container with Clone support
- `AudioSettings` - Master/music/SFX volumes, mute, sound effects toggle
- `DisplaySettings` - Theme, font size, animations, high contrast
- `GameplaySettings` - Difficulty, auto-save, tooltips, hints
- `ControlsSettings` - Global/game keybindings dictionaries

### ViewModels
- `SettingsWindowViewModel` - Main controller with Apply/Cancel/OK
- `AudioSettingsTabViewModel` - Audio settings bindings
- `DisplaySettingsTabViewModel` - Display settings with theme list
- `GameplaySettingsTabViewModel` - Gameplay with difficulty descriptions
- `ControlsSettingsTabViewModel` - Keybinding management with rebind
- `KeybindingViewModel` - Individual keybinding with rebind command

### Views
- Updated `SettingsWindow.axaml` - Tabbed layout with 4 categories

### Styles
- `SettingsStyles.axaml` - Settings-specific styles

### Tests
- 10 new unit tests for tab ViewModels and settings persistence

## Technical Notes
- Real-time change tracking via `HasChanges` property
- Deep clone support for cancel revert
- DifficultyDescription computed property updates on change
- Reset to Defaults clears and repopulates bindings
