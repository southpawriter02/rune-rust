# v0.16.1a Changelog - Proficiency Level Enum

## Summary

Implements the foundational **Proficiency Level Enum** system that defines weapon proficiency levels and their combat effects. This establishes the core data structures and interfaces for the Weapon Proficiency System (v0.16.1).

## Added

### Domain Layer - Enums (2 files)

| File                        | Description                                                                                    |
| --------------------------- | ---------------------------------------------------------------------------------------------- |
| `WeaponProficiencyLevel.cs` | Enum defining 4 proficiency levels: NonProficient (0), Proficient (1), Expert (2), Master (3). |
| `TechniqueAccess.cs`        | Enum defining 4 technique tiers: None (0), Basic (1), Advanced (2), Signature (3).             |

### Domain Layer - Value Objects (1 file)

| File                   | Description                                                                                                   |
| ---------------------- | ------------------------------------------------------------------------------------------------------------- |
| `ProficiencyEffect.cs` | `readonly record struct` encapsulating combat effects per proficiency level. Factory methods with validation. |

### Application Layer - Interfaces (1 file)

| File                            | Description                                                                                   |
| ------------------------------- | --------------------------------------------------------------------------------------------- |
| `IProficiencyEffectProvider.cs` | Interface for proficiency effect data access. Methods for effects, modifiers, and validation. |

### Configuration (2 files)

| File                                             | Description                                                     |
| ------------------------------------------------ | --------------------------------------------------------------- |
| `config/proficiency-effects.json`                | Effect definitions for all 4 proficiency levels with modifiers. |
| `config/schemas/proficiency-effects.schema.json` | JSON Schema draft-07 for configuration validation.              |

### Unit Tests (3 files, 86 test cases)
