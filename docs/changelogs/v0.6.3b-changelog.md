## Summary

Implements intelligent tab completion with pluggable sources, context-aware matches, and popup display for multiple suggestions.

## Key Features

| Feature | Description |
|---------|-------------|
| **Tab Key** | Trigger completion |
| **Context-Aware** | Different sources for command vs target vs item |
| **Single Match** | Complete inline |
| **Multiple Matches** | Show popup for selection |
| **Pluggable Sources** | Register custom completion sources |
| **Case-Insensitive** | Match regardless of case |

---

## Completion Sources

| Source | Commands | Provides |
|--------|----------|----------|
| `CommandCompletionSource` | (first word) | Command names |
| `TargetCompletionSource` | attack, examine, look | Targets |
| `ItemCompletionSource` | use, equip, drop | Inventory items |
| `DirectionCompletionSource` | go, move, walk | Directions |
