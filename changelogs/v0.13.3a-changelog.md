# Changelog - v0.13.3a: Resource Inventory Panel

**Release Date:** 2026-01-21
**Version:** 0.13.3a

## Overview
This version implements the Resource Inventory Panel for the Crafting UI, displaying player resources organized by category with type icons, quantities, and filter tabs.

---

## Added

### DTOs (Presentation Layer)
- `ResourceCategory` enum: Ore, Herb, Leather, Gem, Misc
- `ResourceStackDisplayDto` record: Resource stack for UI display
- `ResourceCategoryDisplayDto` record: Grouped category display

### Configuration
- `ResourcePanelConfig` class: Panel dimensions and category colors
- `config/resource-panel.json`: Runtime configuration
- `config/schemas/resource-panel.schema.json`: JSON Schema

### Renderers
- `ResourceStackRenderer`: Formats resource stacks with icons and quantities
  - `GetTypeIcon()`: Returns [O], [H], [L], [G], [M]
  - `GetTypeColor()`: Category-specific colors
  - `FormatQuantity()`: x## or x#k format
  - `FormatStack()`: Full formatted display
  - `FormatStackWithIcon()`: Icon + name + quantity

### UI Components
- `ResourceInventoryPanel`: Main panel component
  - Categorized column display
  - Filter tabs (All/Ore/Herb/Leather/Gem/Misc)
  - Keyboard input handling (keys 1-6)
  - Overflow indicators for large collections
  - Empty state handling

### Unit Tests (53 new tests)
- `ResourceStackRendererTests`: Icon, color, quantity, stack formatting
- `ResourceInventoryPanelTests`: Filter, input handling, rendering

---

## Technical Notes

### Type Icons
- [O] Ore (mining resources)
- [H] Herb (plant resources)
- [L] Leather (animal hides)
- [G] Gem (precious stones)
- [M] Misc (other materials)

### Quantity Formatting
- Under 1000: `x##` (e.g., x24, x156)
- 1000+: `x#k` (e.g., x1k, x25k)

### Filter Key Bindings
- 1: All (no filter)
- 2: Ore
- 3: Herb
- 4: Leather
- 5: Gem
- 6: Misc
