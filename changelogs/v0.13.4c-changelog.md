# v0.13.4c Changelog: Leaderboard View

## Summary

Implements the Leaderboard View UI component for displaying ranked entries by category with score breakdown and personal best highlighting.

---

## Added

### Configuration
- **`LeaderboardViewConfig.cs`**: Panel dimensions, column widths, and rank colors
- **`config/leaderboard-view.json`**: Default configuration values
- **`config/schemas/leaderboard-view.schema.json`**: JSON schema for validation

### DTOs
- **`LeaderboardDisplayDto.cs`**: Display-ready entry data (rank, name, class, level, score, date, category-specific fields)
- **`ScoreComponentDto.cs`**: Score breakdown with base/multiplier/penalty flags

### UI Components
- **`LeaderboardView.cs`**: Main panel with category tabs (1-5), entry rendering, personal best section
- **`ScoreBreakdownPanel.cs`**: Tree-style component list showing score calculation
- **`PersonalBestHighlight.cs`**: "Your Best" display with "NEW RECORD!" notification

### Renderer
- **`LeaderboardEntryRenderer.cs`**: Column-aligned entry formatting with rank indicators (#1, #2, #3, >N)

### Commands
- **`LeaderboardCommand.cs`**: Command handler for `leaderboard`, `lb`, `scores`, `highscores`

### Tests (50 new tests)
- `LeaderboardEntryRendererTests.cs` - Rank formatting, colors, score/time/date formatting
- `ScoreBreakdownPanelTests.cs` - Component formatting, rendering
- `LeaderboardViewTests.cs` - Category selection, rendering

---

## Technical Notes

### API Adaptations
| Design Spec | Implementation | Reason |
|-------------|----------------|--------|
| `entry.CharacterClass` | `entry.ClassName` | Actual `LeaderboardEntry` property name |
| `GetStatValue()` | `GetStatistic()` | Actual `PlayerStatistics` method name |
| `entry.PlayerId` | Compare by `PlayerName` | `LeaderboardEntry` doesn't have `PlayerId` |

### Category Tab Mapping
| Key | Category |
|-----|----------|
| 1 | High Score |
| 2 | Speedrun |
| 3 | No Death |
| 4 | Achievement Points |
| 5 | Boss Slayer |

### Rank Colors
- #1: Yellow (Gold)
- #2: Gray (Silver)
- #3: DarkYellow (Bronze)
- Current Player: Cyan
- Others: White
