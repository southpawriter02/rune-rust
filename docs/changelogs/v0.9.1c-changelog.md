## Summary

Implements smooth theme transitions with crossfade effects, stinger ducking, and fade in/out for game pause handling.

---

## Added

### Application Layer - Interfaces

| Interface | Description |
|-----------|-------------|
| `ICrossfadeManager` | Transition operations |

**ICrossfadeManager Members:**
- `CrossfadeTo(track, duration, onComplete)` — Track transition
- `CrossfadeToTheme(theme, duration)` — Theme transition
- `FadeOut(duration)` — Fade to silence
- `FadeIn(duration)` — Fade from silence
- `DuckAndPlay(path, duckLevel, onComplete)` — Stinger with ducking
- `Cancel()` — Stop active transition
- `IsTransitioning` — State property

### Infrastructure Layer - Services

| Class | Description |
|-------|-------------|
| `CrossfadeManager` | Transition implementation |

**CrossfadeManager Features:**
- Linear volume interpolation (20 steps)
- Cancellable via CancellationToken
- Stinger ducking with automatic restore
- Intro track support for themes
