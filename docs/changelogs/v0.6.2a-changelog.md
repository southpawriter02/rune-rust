# v0.6.2a Changelog: Combat Grid View

**Version:** 0.6.2a  
**Phase:** Combat Display - Combat Grid View  
**Date:** 2026-01-14  
**Status:** ✅ Complete

---

## Summary

Implements enhanced tactical combat display with 8x8 grid, combatant list with HP/MP bars, and turn information bar. Supports cell highlighting for movement/attack/ability ranges.

---

## Added

### Domain Layer

| Component | File | Description |
|-----------|------|-------------|
| `HighlightType` | Enums/HighlightType.cs | Cell highlighting types (Movement, Attack, Ability, Threatened, Selected) |

### Presentation Layer

| Component | File | Description |
|-----------|------|-------------|
| `CombatGridView` | UI/CombatGridView.cs | 8x8 tactical grid with entities and terrain |
| `CombatantListPanel` | UI/CombatantListPanel.cs | Turn-ordered combatant list with HP/MP |
| `TurnInfoBar` | UI/TurnInfoBar.cs | Round, turn, and action economy display |

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Tactical Grid** | 8x8 grid with A-H / 1-8 coordinates |
| **Entity Symbols** | `@`=Player, `M`=Monster, `A`=Ally |
| **Terrain Symbols** | `.`=Open, `~`=Water, `#`=Wall, `,`=Difficult |
| **Cell Highlighting** | Movement (blue), Attack (red), Ability (purple), Selected (yellow) |
| **HP/MP Bars** | Health and mana bars per combatant |
| **Turn Indicator** | `>` marks current turn in combatant list |
| **Action Economy** | Action/Move/Bonus action availability display |
