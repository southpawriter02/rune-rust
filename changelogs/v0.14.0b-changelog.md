# v0.14.0b Changelog - Class Schema

**Version:** 0.14.0b
**Date:** 2026-01-21
**Theme:** Character Configuration Schemas - Class Schema

---

## Overview

This version introduces the Class Schema (`classes.schema.json`), defining the JSON Schema validation for character class configurations. The schema enables IDE autocomplete and validation for class definitions with complex stat modifiers and growth rates.

---

## Added

### Configuration Schemas
- **`config/schemas/classes.schema.json`** - JSON Schema Draft-07 for class definitions
  - Validates `ClassDefinition` objects with all required fields
  - Validates `StatModifiers` for base stat adjustments (positive/negative allowed)
  - Validates `GrowthRates` with non-negative constraints (minimum: 0)
  - Enforces resource type enum: `mana`, `rage`, `energy`, `focus`, `faith`, `stamina`
  - Enforces kebab-case pattern for `id`, `archetypeId`, and ability IDs
  - Supports optional `requirements` object with level, achievements, and quest conditions

### Unit Tests
- **`ClassSchemaTests.cs`** - 5 new unit tests
  - `Schema_IsValidJsonSchemaDraft07` - Validates schema structure
  - `ExistingClassesJson_ValidatesAgainstSchema` - Validates existing configuration
  - `ClassId_InvalidPattern_FailsValidation` - Tests ID pattern enforcement
  - `PrimaryResourceId_InvalidValue_FailsValidation` - Tests resource type enum
  - `GrowthRates_NegativeValue_FailsValidation` - Tests growth rate constraints

## Changed

### Configuration Files
- **`config/classes.json`** - Added `$schema` reference for IDE validation

## Technical Details

### Schema Definitions

| Definition | Purpose |
|------------|---------|
| `ClassDefinition` | Validates class objects with stats and abilities |
| `StatModifiers` | Validates 8 stat modifiers (can be positive or negative) |
| `GrowthRates` | Validates 8 per-level growth values (must be >= 0) |

### Resource Type Enum

| Resource | Typical Class |
|----------|---------------|
| `mana` | Galdr-Caster |
| `rage` | Shieldmaiden |
| `energy` | Shadow-Walker |
| `focus` | Scrap-Tinker |
| `faith` | Blood-Priest |
| `stamina` | Generic martial |
