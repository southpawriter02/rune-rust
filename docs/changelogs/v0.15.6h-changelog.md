# v0.15.6h - Specialization Integration

**Release Date:** 2026-01-26

## Summary

Implements enums, value objects, and interface for **specialization-specific perception abilities**.

## Added

### Enums (Domain)

| Enum                    | Values                                                                                      |
| ----------------------- | ------------------------------------------------------------------------------------------- |
| `PerceptionAbilityType` | PassivePerception, Examination, Investigation, Search                                       |
| `ConditionType`         | ObjectCategory, ObjectType, ElementType, InvestigationTarget, Proximity, Always, ObjectTags |
| `PerceptionEffectType`  | DiceBonus, PassiveBonus, AutoSuccessLayer, AutoDetect, BonusLore, InvestigationBonus        |

### Value Objects (Domain)

| Value Object                  | Purpose                                      |
| ----------------------------- | -------------------------------------------- |
| `AbilityCondition`            | Conditions for ability activation            |
| `PerceptionAbilityEffect`     | Effects when ability activates               |
| `PerceptionAbility`           | Specialization perception ability definition |
| `PerceptionAbilityActivation` | Tracking ability usage                       |
| `InvestigationModifiers`      | Investigation bonuses from specialization    |

### Interfaces (Application)

| Interface                          | Purpose                                          |
| ---------------------------------- | ------------------------------------------------ |
| `ISpecializationPerceptionService` | Manages perception abilities and applies bonuses |
