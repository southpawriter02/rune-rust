### Features

#### Equipment Slots
- [x] **Equipment Slots** - Weapon, Armor, Shield, Helmet, Boots, Ring, Amulet
- [x] **Equip Command** - Put on equipment (`equip <item>`)
- [x] **Unequip Command** - Remove equipment (`unequip <slot>`)
- [x] **Equipment Display** - Show equipped items in status/inventory

#### Equipment Stats
- [x] **Equipment Stats** - Items modify player stats when equipped
- [x] **Weapon Damage Dice** - Weapons define damage dice pools (integrates with v0.0.5c `GetPlayerDamagePool`)
- [x] **Armor Defense** - Armor provides defense bonuses

#### Weapon Types
- [x] **Swords** - Balanced damage dice (1d8)
- [x] **Axes** - Higher damage (1d10), lower accuracy (-1 attack)
- [x] **Daggers** - Lower damage (1d4), bonus to Finesse (+2)
- [x] **Staffs** - Magic-focused (1d6), bonus to Will (+2)

#### Armor Types
- [x] **Light Armor** - Low defense (1-3), no penalties
- [x] **Medium Armor** - Moderate defense (4-5), minor initiative penalty (-1 to -2)
- [x] **Heavy Armor** - High defense (6+), significant initiative penalty (-3), attribute requirements

#### Equipment Requirements
- [x] **Stat Requirements** - Minimum attribute values to equip items
- [x] **Class Requirements** - Certain equipment restricted by class
