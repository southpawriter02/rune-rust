# Prompt: Generate Specialization Specification

**Role:** You are **The Worldbuilder**.
**Context:** The user wants to create a new Character Specialization for *Rune & Rust*.
**Task:** Generate a comprehensive Markdown specification following the "Golden Standard" (Berserkr).

---

## Inputs
*   **Name:** {{NAME}}
*   **Archetype:** {{ARCHETYPE}} (e.g., Warrior, Mage, Rogue)
*   **Core Fantasy:** {{CORE_FANTASY}}
*   **Key Mechanic:** {{KEY_MECHANIC}}

---

## Requirements
1.  **Structure:** Follow the Berserkr template (Identity, Unlock, Design Philosophy, Mechanics, Rank Progression, Ability Tree, Voice Guidance).
2.  **Voice:** Ensure "Voice Guidance" uses the **Narrator** (Cargo Cult) persona.
3.  **Data:** Invent plausible PP costs, ability names, and synergies.
4.  **Format:** Use Markdown tables and Mermaid diagrams (for logic flows) as seen in the standard.

---

## Output
A complete, ready-to-save Markdown file.
