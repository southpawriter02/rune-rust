## v0.1.3 - Advanced Procedural Architecture

**Focus:** Coherent, varied dungeon layouts with architectural intelligence

### Features
- [ ] **Architectural Styles** - Distinct construction patterns:
  - Natural Caves (organic, irregular, no doors)
  - Carved Halls (rectangular, orderly, pillars)
  - Ruined Structures (collapsed, rubble, gaps)
  - Ancient Temples (grand chambers, symmetry)
- [ ] **Room Shape Variety** - Non-rectangular rooms:
  - Circular (ritual rooms, arenas)
  - L-shaped (ambush opportunities)
  - Irregular caverns (natural formations)
- [ ] **Landmark Rooms** - Hand-crafted centerpieces:
  - Boss arenas with specific architecture
  - Unique puzzle chambers
  - Safe haven designs
- [ ] **Structural Coherence** - Logical architecture:
  - Corridors connect rooms in carved areas
  - Support structures in large rooms
  - Consistent materials per depth/biome
- [ ] **Dungeon Age & Decay**:
  - Fresh (intact, occupied)
  - Ancient (worn, dusty)
  - Collapsed (rubble, blocked passages)
  - Reclaimed (nature overtaking)

### Customization Features
- [ ] **Architectural Style Definitions** - JSON-configured styles
- [ ] **Room Template Library** - Expandable template collection
- [ ] **Coherence Rule Definitions** - Configurable structural logic
- [ ] **Landmark Room Pool** - Curated special room designs
