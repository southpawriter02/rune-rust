# v0.13.0d Changelog: Defense & Stance Display

**Release Date:** 2026-01-20

---

## Summary

Implemented Defense Action Prompts and Stance Indicator UI components for displaying defense options during attacks and active combat stance with modifiers.

---

## Added

### DTOs

- **DefensePromptDto** — Defense action option display
- **DefenseResultDto** — Defense result feedback  
- **IncomingAttackDto** — Incoming attack information
- **ReactionWindowDto** — Reaction timing window
- **ModifierDisplayDto** — Stat modifier display
- **StanceDisplayDto** — Active stance display
- **StanceSwitchDto** — Stance transition comparison

### Configuration

- **DefenseDisplayConfig** — Defense prompt configuration
  - `DefenseActionKeyConfig` — Key mappings (B, D, P, C)
  - `DefenseSymbolConfig` — Timing bar symbols
  - `DefenseColorConfig` — Result and timing colors
  - `DefenseDisplaySettings` — Behavior settings

- **StanceDisplayConfig** — Stance indicator configuration
  - `StanceIconConfig` — Category icons (Unicode/ASCII)
  - `StanceColorConfig` — Category and modifier colors
  - `StanceDisplaySettings` — Display behavior

### Renderers

- **DefenseRenderer**
  - Timing bar formatting (progress visualization)
  - Action key mapping
  - Result color mapping (success/failure)
  - Damage reduction formatting

- **StanceRenderer** 
  - Stance icon mapping (5 categories + fallback)
  - Modifier formatting (+/- values, percentages)
  - Switch comparison with change indicators

### Configuration Files

- `config/defense-display.json`
- `config/stance-display.json`

### Tests

- `DefenseRendererTests` (15 tests)
- `StanceRendererTests` (18 tests)
