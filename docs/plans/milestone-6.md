# Milestone 6: The Weaver (Magic Systems)

This phase implements the "Weaving" mechanics, distinguishing Aetheric manipulation from standard combat abilities through risk-based systems and runic logic.

	•	[ ] v0.4.3: Magic Core
	◦	Scope: Implement AetherService for advanced flux tracking, the Spell entity (distinct from ActiveAbility), and the "Weaving" mechanic which replaces standard cooldowns with escalating risk.
	•	[ ] v0.4.4: Spells & Chants
	◦	Scope: Implement the cast command, multi-turn Chant mechanics (interrupters/concentration), and seed the initial Mystic spell lists (e.g., Aether Dart, Mind Spike).
	•	[ ] v0.4.5: Runic Inscription
	◦	Scope: Implement the full Runeforging crafting trade, Rune items (consumable/socketable), and equipment embossing logic that applies ItemProperty modifiers.
	•	[ ] v0.4.6: The Glitch (Wild Magic)
	◦	Scope: Implement WildMagicService to handle critical spell failures, calculating "Paradox" events based on current Corruption/Stress levels and biome instability.
	•	[ ] v0.4.7: Magic UI
	◦	Scope: Create the Grimoire UI tab for spell management and add TUI visual feedback for active Chants (e.g., progress bars) and inscribed Rune effects.
