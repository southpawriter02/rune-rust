## Summary

Implements text-based combat animations with frame-based playback, animation queue, template variable substitution, and speed controls.

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Frame Playback** | Sequential frame display with timing |
| **Queue System** | Multiple animations queued for playback |
| **Variable Substitution** | `{actor}`, `{target}`, `{value}`, `{verb}` |
| **Speed Control** | `SpeedMultiplier` (0.5x to 2x) |
| **Skip Support** | `SkipCurrent()` and `ClearQueue()` |
| **Color Support** | Per-frame foreground colors |
