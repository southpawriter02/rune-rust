## Service Integration

The panel integrates with existing TUI services:
- **`ICommandHistoryService`** - Stores and recalls previous commands
- **`ITabCompletionService`** - Provides contextual suggestions
- **`CommandSubmitted` event** - Notifies when command is submitted
