# File Locations Quick Reference

| File Type               | Location                          |
|-------------------------|-----------------------------------|
| Terminology config      | `config/terminology.json`         |
| Lexicon config          | `config/lexicon.json`             |
| Theme presets           | `config/themes.json`              |
| Environment descriptors | `config/descriptors/environmental.json` |
| Combat descriptors      | `config/descriptors/combat.json`  |
| Race definitions        | `config/races.json`               |
| Class definitions       | `config/classes.json`             |
| Ability definitions     | `config/abilities.json`           |
| Resource types          | `config/resources.json`           |
| Monster definitions     | `config/monsters.json`            |
| JSON schemas            | `config/schemas/*.schema.json`    |
