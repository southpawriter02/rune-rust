# Changelog: v0.17.0a - Lineage Enum & Definition

**Date:** 2025-01-29

## Summary

Implements the foundational Lineage system for the v0.17.x Character Creation System. Lineage represents a character's bloodline heritage and is the first permanent choice in character creation. This version introduces the core data structures for attribute modifiers.

## Added

### Domain Layer

- **Lineage** (`Enums/Lineage.cs`)
    - Enum with 4 bloodline heritage values
    - `ClanBorn` (0), `RuneMarked` (1), `IronBlooded` (2), `VargrKin` (3)
    - Extensive XML documentation with lore and gameplay effects

- **CoreAttribute** (`Enums/CoreAttribute.cs`)
    - Enum with 5 core character attributes
    - `Might` (0), `Finesse` (1), `Wits` (2), `Will` (3), `Sturdiness` (4)
    - Required for point-buy system and lineage modifiers

- **LineageAttributeModifiers** (`ValueObjects/LineageAttributeModifiers.cs`)
    - Readonly record struct with 7 properties
    - Properties: `MightModifier`, `FinesseModifier`, `WitsModifier`, `WillModifier`, `SturdinessModifier`, `HasFlexibleBonus`, `FlexibleBonusAmount`
    - Computed property: `TotalFixedModifiers`
    - Methods: `GetModifier(CoreAttribute)`, `GetEffectiveModifier(CoreAttribute, CoreAttribute?)`
    - Static factory properties: `ClanBorn`, `RuneMarked`, `IronBlooded`, `VargrKin`

- **LineageDefinition** (`Entities/LineageDefinition.cs`)
    - Entity implementing `IEntity`
    - Properties: `Id`, `LineageId`, `DisplayName`, `Description`, `SelectionText`, `AttributeModifiers`, `AppearanceNotes`, `SocialRole`
    - Factory method: `Create(...)` with argument validation and logging
    - Helper methods: `GetTotalFixedModifiers()`, `RequiresFlexibleBonusSelection()`

### Configuration Files

- **lineages.json** (`config/lineages.json`)
    - Complete definitions for all 4 lineages
    - Includes display metadata, attribute modifiers, appearance notes, social roles

- **lineages.schema.json** (`config/schemas/lineages.schema.json`)
    - JSON Schema draft-07 for validation
    - Defines `lineageDefinition` and `attributeModifiers` structures
