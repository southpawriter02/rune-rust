# v0.15.6f - Search Command Enhancement

**Release Date:** 2026-01-26

## Summary

Implements value objects and interfaces for **Search Command Enhancement** with active perception bonus (+2) and time-based searching.

## Added

### Value Objects (Domain)

| Value Object         | Purpose                                      |
| -------------------- | -------------------------------------------- |
| `ActiveSearchResult` | Comprehensive search result with discoveries |
| `FoundItem`          | Discovered item with location and DC info    |
| `SearchParameters`   | Search operation configuration               |

### Enums (Domain)

| Enum           | Values                  |
| -------------- | ----------------------- |
| `SearchTarget` | Room, Container, Object |

### Interfaces (Application)

| Interface              | Purpose                                |
| ---------------------- | -------------------------------------- |
| `ISearchService`       | Active search with +2 perception bonus |
| `ISearchConfiguration` | Search time and bonus settings         |
| `ISearchRenderer`      | Search result display formatting       |

## Key Features

### ActiveSearchResult VO

- Fields: `RoomId`, `CharacterId`, `WitsCheckResult`, `TimeSpent`
- Collections: `ItemsFound`, `HiddenElementsRevealed`, `ContainersSearched`, `HintsDiscovered`
- Helpers: `HasDiscoveries`, `TotalDiscoveries`, `HasItems`

### ISearchService Interface

- `SearchAsync()` with +2 active search bonus
- `CalculateSearchTime()` by room size
- `GetActivePerceptionValue()` including bonus

### Search Time by Room Size

| Size   | Time   |
| ------ | ------ |
| Small  | 5 min  |
| Medium | 10 min |
| Large  | 15 min |
| XLarge | 20 min |
