## Summary

Implements a persistent status bar in the footer panel that displays player HP, resources, XP, gold, and location. Features three format modes (Compact/Expanded/Combat) and auto-refresh on data changes via the reactive update pattern.

|-----------|------|-------------|
| `StatusBar` | UI/StatusBar.cs | Status bar with 3 format modes |

---

## Key Features

| Feature | Description |
|---------|-------------|
| Compact Format | For terminals <100 width: `HP:X/Y \| MP:X/Y \| XP:X/Y \| Gold:N` |
| Expanded Format | For terminals ≥100 width: Full labels with Location |
| Combat Format | Shows round number and turn indicator |
| Low HP Warning | ⚠ symbol when HP ≤ 25% |
| Auto-Refresh | Subscribes to `OnDataChanged` event |
| Resource Abbreviations | Mana→MP, Rage→RG, Energy→EN, etc. |
