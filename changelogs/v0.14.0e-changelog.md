## Overview

This version introduces the Progression Schema (`progression.schema.json`) completing the Character Configuration Schemas series. The schema validates XP curves, level milestones, terminology customization, and stat bonuses for level progression.

---

## Added

### Configuration Schemas
- **`config/schemas/progression.schema.json`** - JSON Schema Draft-07 for progression configuration
  - Validates `maxLevel` range (1-100)
  - Validates `curveType` enum (Linear, Exponential, Custom)
  - Validates `baseXpRequirement` minimum (≥1)
  - Validates `xpMultiplier` minimum (≥1.0)
  - Includes `StatBonuses` definition (maxHealth, attack, defense)
  - Includes `LevelMilestone` definition with title and customRewards

### Unit Tests

## Technical Details

### XP Curve Types
| Curve Type | Formula | Use Case |
|------------|---------|----------|
| `Linear` | baseXP × level | Predictable progression |
| `Exponential` | baseXP × (multiplier ^ level) | Extended end-game |
| `Custom` | Per-milestone xpRequired | Full control |

### StatBonuses Object
| Property | Type | Range | Description |
|----------|------|-------|-------------|
| `maxHealth` | integer | ≥0 | Health increase per level |
| `attack` | integer | ≥0 | Attack increase per level |
| `defense` | integer | ≥0 | Defense increase per level |

### LevelMilestone Object
| Property | Required | Description |
|----------|----------|-------------|
| `level` | Yes | Level number (≥1) |
| `title` | No | Title earned (e.g., "Veteran") |
| `customRewards` | No | Array of reward strings |
| `statOverrides` | No | Override default stat bonuses |
| `xpRequired` | No | Custom XP for this level |
