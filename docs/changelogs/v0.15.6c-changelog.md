# v0.15.6c - Object Category Descriptors

**Release Date:** 2026-01-26

## Summary

Implements the **Object Category Descriptors** Domain entity and Application repository interface. This version adds the data layer for category-based examination text with biome affinity and weighted variant selection.

## Added

### Enums (Domain)

| Addition                       | Description                                           |
| ------------------------------ | ----------------------------------------------------- |
| `ObjectCategory.Furniture`     | Functional furniture (Table, Chair, Bed, Workbench)   |
| `ObjectCategory.Environmental` | Room structural features (Wall, Floor, Ceiling, Vent) |
| `ObjectCategory.Creature`      | Living or deceased creatures and remains              |

### Entities (Domain)

| Entity             | Purpose                                             |
| ------------------ | --------------------------------------------------- |
| `ObjectDescriptor` | Stores layered examination text with biome affinity |

### Interfaces (Application)

| Interface                     | Purpose                                        |
| ----------------------------- | ---------------------------------------------- |
| `IObjectDescriptorRepository` | Repository contract for descriptor data access |

## Key Features

### ObjectDescriptor Entity

- Properties: `DescriptorId`, `ObjectCategory`, `ObjectType`, `Layer`, `RequiredDc`
- Biome affinity: `BiomeAffinity`, `IsUniversal`, `MatchesBiome()`
- Puzzle integration: `RevealsHint`, `RevealsSolutionId` (for v0.15.6e)
- Weighted selection: `Weight` property for randomized variety
- Factory method: `Create()` with validation

### Layer DC Mapping

| Layer | Name     | Required DC |
| ----- | -------- | ----------- |
| 1     | Cursory  | 0 (auto)    |
| 2     | Detailed | 12          |
| 3     | Expert   | 18          |
