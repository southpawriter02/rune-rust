# v0.16.1b Changelog - Weapon Category System

## Summary

Implements the **Weapon Category System** that classifies weapons into 11 distinct categories for proficiency tracking. Each category has an associated primary attribute (MIGHT, FINESSE, WILL, or WITS) for attack/damage calculations.
