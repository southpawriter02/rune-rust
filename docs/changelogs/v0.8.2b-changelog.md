# v0.8.2b Changelog - Dialogue Window

## Added

### Models
- `DialogueAction` enum (None, AcceptQuest, DeclineQuest, OpenShop, Leave, GiveItem, TakeItem)
- `DialogueNode` record for dialogue tree nodes
- `DialogueChoiceData` record for choice data

### ViewModels
- `DialogueChoiceViewModel` - Choice display with action badges
- `DialogueWindowViewModel` - Typing animation and branching

### Views
- `DialogueWindow.axaml` - Portrait and choices layout
- `DialogueWindow.axaml.cs` - Keyboard support

### Styles
- `DialogueWindowStyles.axaml` - NPC name and choice styling

### Features
| Feature | Description |
|---------|-------------|
| Typing Animation | Character-by-character reveal (30ms default) |
| Skip Animation | Click or Space/Enter to skip |
| Branching Dialogue | Navigate tree via choices |
| Action Badges | [ACCEPT QUEST], [SHOP], [LEAVE] |
| Keyboard Support | 1-9 for choices, Escape to close |
