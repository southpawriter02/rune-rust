**Version:** 0.19.1
**Theme:** Midgard (The Tamed Ruin)
**Prerequisites:** v0.19.0 (Core Biome Infrastructure)

---

## Executive Summary

v0.19.1 implements Midgard, the starting realm and agricultural heartland of Aethelgard. This version defines the complete Midgard biome configuration including all four canonical sub-zones from the lore (The Greatwood, The Asgardian Scar, The Souring Mires, The Serpent Fjords), faction spawn pools, environmental conditions, zone-specific hazards, terrain types, and room templates.

Midgard serves as the baseline realm against which all other realms are measured.

The work is divided into **2 sub-phases**:

| Phase    | Name                   | Focus                                         | Est. Tests |
| -------- | ---------------------- | --------------------------------------------- | ---------- |
| v0.19.1a | Midgard Core           | Biome definition, zones, base conditions      | ~18        |
| v0.19.1b | Midgard Content        | Hazards, terrain, templates, enemies, resources | ~14       |

## v0.19.1a: Midgard Core Implementation

[v0.19.1a Design Specification](v0.19.1a-design-specification.md)

### Overview

This phase implements the complete Midgard realm with its four canonical sub-zones from the lore, faction spawn pools, and environmental conditions.

### Scope

**In Scope:**

- Midgard `BiomeDefinition` with complete properties
- Four canonical sub-zones from lore:
  - **The Greatwood** — Feral forest, MutagenicSpores (light)
  - **The Asgardian Scar** — Blight epicenter, BlightStorm + RealityFlux
  - **The Souring Mires** — Toxic wetland, ToxicAtmosphere (light)
  - **The Serpent Fjords** — Poisoned coast, Leviathan hazard
- Faction spawn pools per zone (from lore §7)
- Zone property overrides and DC modifiers
- Lore integration (quotes, descriptions, color palettes)

**Out of Scope:**

- Zone-specific hazards (v0.19.1b)
- Terrain types (v0.19.1b)
- Room templates (v0.19.1b)
- Enemy definitions (v0.19.1b)
- Resource tables (v0.19.1b)

### Key Deliverables

| Type             | Count | Details                                        |
| ---------------- | ----- | ---------------------------------------------- |
| Configuration    | 1     | midgard.json (biome definition + zones)        |
| Zone Definitions | 4     | Greatwood, Scar, Mires, Fjords                 |
| Faction Pools    | 4     | One per zone                                   |
| Unit Tests       | ~18   |                                                |
