# v0.6.1a Changelog: Health Bars

**Date:** 2026-01-14

## Summary

Implements visual bar components that display HP, MP, and other resources as filled bar graphics. Bars dynamically change color based on percentage thresholds (Green→Yellow→Red) providing instant visual feedback about player status.

---

## Added

### Presentation Layer

| Component | File | Description |
|-----------|------|-------------|
| `BarStyle` | UI/BarStyle.cs | Enum (4 display styles) |
| `BarType` | UI/BarType.cs | Enum (5 bar categories) |
| `ColorThreshold` | Configuration/ColorThreshold.cs | Threshold percentage + color record |
| `BarCharacters` | Configuration/ColorThreshold.cs | Unicode/ASCII character config |
| `HealthBarConfig` | Configuration/HealthBarConfig.cs | Full bar configuration |
| `HealthBar` | UI/HealthBar.cs | Core bar rendering component |
| `ResourceBar` | UI/ResourceBar.cs | Resource-specific bar wrapper |

---

## Key Features

| Feature | Description |
|---------|-------------|
| **4 Display Styles** | Standard `████░░`, Detailed `[████] 80/100`, Compact `████80`, Numeric `80/100` |
| **5 Bar Types** | Health, Mana, Experience, Stamina, Custom |
| **Color Thresholds** | Health: Green→Yellow→Red→DarkRed by % |
| **Unicode/ASCII** | Auto-fallback: `█░` → `#-` |
| **Labeled Rendering** | `HP: ████████ 80/100` |
