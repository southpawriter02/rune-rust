# Data & Content Directory

This directory contains sample data, content templates, and reference material for writers creating game content.

---

## Directory Structure

```
docs/09-data/
├── overview.md                    ← You are here
├── sample-codex-entries.md        ← Sample Codex entries with Data Captures
├── sample-bestiary-entries.md     ← Sample Bestiary entries with discovery levels
├── sample-alchemy-items.md        ← Sample alchemical items with recipes
└── fragment-placement-guide.md    ← Guidelines for placing captures in world
```

---

## Purpose

The `09-data` directory serves as:

1. **Writer Reference** — Templates and examples for creating new content
2. **Style Guide** — Consistent voice and formatting across entries
3. **Content Inventory** — Catalog of existing lore and creature entries
4. **Integration Guide** — How content ties into game systems

## Content Creation Workflow

1. **Research** — Check lexicon and background files for existing lore
2. **Outline** — Define entry structure and fragment count
3. **Write Fragments** — Create individual Data Captures
4. **Define Placement** — Determine where each fragment is found
5. **Set Thresholds** — Define keyword unlocks and rewards
6. **Cross-Reference** — Link to related entries

---

## Quality Checklist

| Criterion | Description |
|-----------|-------------|
| **Voice Consistency** | Matches capture type voice (see samples) |
| **Lore Accuracy** | Consistent with established setting |
| **Progressive Reveal** | Early fragments hint, later reveal |
| **Earned Discovery** | Important info requires effort |
| **Cross-Reference** | Mentions related entries |
| **Mystery Preserved** | Leaves questions for exploration |
