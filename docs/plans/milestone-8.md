# Milestone 8: The Adversary (Advanced Combat)

This phase focuses on deepening tactical complexity with advanced enemy behaviors, boss mechanics, and squad-based features.

	•	[ ] v0.6.0: Enemy Traits (Affixes)
	◦	Scope: Implement CreatureTraitService to generate Elite enemies with procedural modifiers (e.g., "Explosive," "Teleporting") and stat scaling.
	•	[ ] v0.6.1: Advanced AI
	◦	Scope: Implement BehaviorTree AI to support complex squad tactics, including healing allies, flanking maneuvers, and coordinated fleeing.
	•	[ ] v0.6.2: Boss Mechanics
	◦	Scope: Implement BossService to handle multi-phase encounters, phase transitions, and "Legendary Actions" that occur outside the standard turn order.
	•	[ ] v0.6.3: Stealth & Ambush
	◦	Scope: Implement StealthService, the hide command, and "Surprise Round" mechanics for initiating combat with an advantage.
	•	[ ] v0.6.4: Companion System
	◦	Scope: Implement Companion entities, recruitment logic via Faction standing, and a Squad Command UI for directing allies in combat.
