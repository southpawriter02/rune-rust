**Version:** 0.9.0b  
**Phase:** Audio Settings Integration  
**Date:** 2026-01-16

## Summary

Integrates the audio service with the settings UI, providing functional volume sliders, mute controls, percentage display, and settings save/load/revert.

---

## Added

### Application Layer - DTOs

| DTO | Description |
|-----|-------------|
| `AudioSettings` | Persistence model for audio configuration |

**AudioSettings Properties:**
- `Enabled` — Global audio toggle
- `MasterVolume`, `MusicVolume`, `EffectsVolume`, `UiVolume` — 0.0-1.0
- `Muted` — Global mute state
- `TuiBellEnabled` — Console bell toggle

### Presentation Layer - ViewModels

| ViewModel | Description |
|-----------|-------------|
| `AudioSettingsViewModel` | Settings tab ViewModel |

**AudioSettingsViewModel Features:**
- Observable volume properties with real-time updates
- Percentage display properties (e.g., "80%")
- IAudioService integration for immediate feedback
- `LoadSettings()`, `GetSettings()`, `SaveSettings()`, `RevertSettings()`

### Presentation Layer - Views

| View | Description |
|------|-------------|
| `AudioSettingsTab.axaml` | Audio settings UI |

**UI Elements:**
- Enable Audio checkbox
- Volume sliders (Master, Music, Effects, UI)
- Percentage labels
- Mute All checkbox
- TUI Bells checkbox
