# v0.7.3a Changelog - Combat Grid Panel

## Overview
Implements the Combat Grid Panel for the Avalonia GUI, providing a tactical battlefield display during combat encounters.

---

## Added

### Enums
- **`GridCellType`** - Types of terrain cells for GUI display (Floor, Wall, Water, Hazard, Cover)

### ViewModels
- **`GridCellViewModel`** - View model for individual grid cells with position, terrain type, walkability, and highlight state
- **`EntityTokenViewModel`** - View model for entity tokens on the grid with symbol, color, and canvas positioning
- **`CombatGridPanelViewModel`** - Main view model managing the grid, tokens, coordinate labels, and legend items
- **`GridLegendItem`** - Record for combat grid legend entries with symbol, label, and color

### Controls
- **`GridCellControl`** - Templated control for rendering grid cells with terrain-based styling
- **`EntityTokenControl`** - Templated control for rendering entity tokens with faction-based colors
- **`CombatGridPanel`** - User control displaying the complete combat grid with coordinate labels and legend

### Styles
- **`CombatGridStyles.axaml`** - Class-based styling for grid cells, entity tokens, and coordinate labels
