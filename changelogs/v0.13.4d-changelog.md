# v0.13.4d Changelog: Dice History Panel

## Summary

Implements the Dice History Panel UI components for displaying comprehensive roll history, statistics, luck rating, streaks, and roll distribution visualization.

---

## Added

### DTOs
- **`DiceHistoryDisplayDto.cs`**: Complete display data for dice history panel (recent rolls, statistics, streaks, distribution)
- **`DiceStatisticsDto.cs`**: Statistics section display data with counts, percentages, and deviations
- **`RollDistributionDto.cs`**: Distribution chart data with outlier detection methods

### Configuration
- **`DiceHistoryPanelConfig.cs`**: Panel dimensions, luck thresholds, colors, bar chart characters
- **`config/dice-history-panel.json`**: Default configuration values
- **`config/schemas/dice-history-panel.schema.json`**: JSON schema for validation

### UI Components
- **`DiceHistoryPanel.cs`**: Main panel orchestrating all sections (recent rolls, statistics, luck rating, streaks, distribution)
- **`StreakIndicator.cs`**: Current and longest streak display with descriptions
- **`RollDistributionChart.cs`**: ASCII bar chart with outlier highlighting

### Renderer
- **`DiceRollRenderer.cs`**: Roll formatting with critical highlighting ("20!", "1!"), color mapping

### Tests (~20 new tests)
- `DiceRollRendererTests.cs` - FormatRoll, FormatRollList, GetRollColor, IsCritical
- `StreakIndicatorTests.cs` - FormatStreak, GetStreakDescription, GetStreakColor, rendering
- `RollDistributionChartTests.cs` - FormatBar, outlier detection, rendering
- `DiceHistoryPanelTests.cs` - Constructor, configuration, Render, ShowStatistics, ShowLuckRating

## Technical Notes

### Dependencies
| Component | Dependency |
|-----------|------------|
| `DiceHistoryPanel` | Consumes `DiceHistoryDisplayDto`, orchestrates sub-components |
| `DiceRollRenderer` | Uses `DiceHistoryPanelConfig` for colors |
| All UI Components | Use `ITerminalService` for output |
| All Components | Integrate with v0.12.0b `IDiceHistoryService` |

### Luck Rating Thresholds
| Rating | Threshold |
|--------|-----------|
| VERY LUCKY | >= +10% |
| LUCKY | >= +5% |
| NORMAL | -5% to +5% |
| UNLUCKY | <= -5% |
| VERY UNLUCKY | <= -10% |

### Color Mapping (Roll Values)
| Roll Value | Color |
|------------|-------|
| 20 | Yellow (Critical Success) |
| 15-19 | Green (Good) |
| 11-14 | White (Average) |
| 6-10 | Gray (Below Average) |
| 2-5 | DarkYellow (Poor) |
| 1 | Red (Critical Failure) |

### Streak Indicators
- Positive streak (lucky): Green, formatted as "+N"
- Negative streak (unlucky): Red, formatted as "-N"
- Zero streak: White, formatted as "0"

### Distribution Bar Chart
- 20 horizontal bars (one per roll value)
- Max bar width: 20 characters (configurable)
- Filled char: `#`, Empty char: `.`
- Outlier threshold: >150% or <50% of expected count
