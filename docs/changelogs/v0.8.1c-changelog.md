# v0.8.1c Changelog - Tooltip System

## Added

### Models
- `TooltipLine` - Label/value pair with optional color
- `TooltipSection` - Header with lines
- `TooltipContent` - Full tooltip with factory methods

### Services
- `ITooltipService` - Interface for tooltip management
- `TooltipService` - Implementation with delayed appearance

### ViewModels
- `TooltipViewModel` - Binds to tooltip service events

### Controls
- `TooltipPanel` - Rich content display with sections

### Styles
- `TooltipStyles.axaml` - Tooltip styling

### Features
| Feature | Description |
|---------|-------------|
| Delayed Appearance | 500ms default delay (configurable) |
| Rich Content | Title, subtitle, sections, description, footer |
| Factory Methods | `Simple()`, `ForItem()`, `ForAbility()`, `ForStatusEffect()` |
| Enable/Disable | Can be disabled via `IsEnabled` property |
| Event-Based | `OnTooltipChanged` event for UI updates |
