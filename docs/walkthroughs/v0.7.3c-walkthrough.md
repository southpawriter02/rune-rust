# v0.7.3c Walkthrough - Combat UI Components

## Overview
This version implements the Combat UI Components: the CombatantListPanel displaying combatants in turn order, and the CombatActionBar providing interactive combat actions.

## Components

### CombatantListPanel
Displays all combatants sorted by initiative:
- **Turn indicator** (â–º) shows current turn
- **HP bars** via `HealthBarControl`
- **Status effects** as icon badges with tooltips
- Color-coded names (green: player, red: enemies)

### CombatActionBar
Shows turn information and action buttons:
- **Turn Info**: Round number, turn indicator, action/bonus points, movement
- **Buttons**: Attack, Move, Ability, Defend, Item, Wait, End Turn
- Commands integrate with `IGridInteractionService` for targeting modes
