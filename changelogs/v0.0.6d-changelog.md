# v0.0.6d Changelog - Status Effect Integration & Polish

**Date:** 2026-01-08

## Added

### Domain Layer - Enums
- `EffectTrigger` enum: OnHit, OnDamage, OnTurnStart, Manual, etc.
- `CleanseType` enum: AllNegative, Physical, Magical, Elemental, Specific

### Domain Layer - Value Objects
- `EffectInteraction` for elemental combos (Wet + Lightning)
- `EffectInteractionResult` for interaction outcomes
- `TurnStartEffectResult` for turn start effect processing
- `AttackEffectResult` for attack effect processing
- `CleanseResult` for cleanse outcomes
- `CleanseItem` for cleanse consumables with factory methods

### Application Layer - Interfaces
- `IEffectInteractionRepository` for interaction definitions
- `ICleanseItemRepository` for cleanse item definitions

### Application Layer - Services
- `EffectInteractionService` for processing elemental combos
- `CleanseService` for effect removal by category

### Infrastructure Layer
- `JsonEffectInteractionRepository` for loading interactions from JSON
- `JsonCleanseItemRepository` for loading cleanse items from JSON

### Configuration
- `config/effect-interactions.json` with 5 interactions:
  - Wet + Lightning → +50% damage, Stunned
  - Wet + Ice → Frozen, remove Wet
  - Burning + Wet → Remove Burning
  - Chilled + Ice → Frozen, remove Chilled
  - On Fire + Wet → Remove On Fire
- `config/cleanse-items.json` with 6 cleanse items
