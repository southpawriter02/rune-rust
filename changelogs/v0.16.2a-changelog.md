# v0.16.2a Changelog - Armor Proficiency Level Enum

## Summary

Implements the foundational **Armor Proficiency Level Enum** system that defines armor proficiency levels and their penalty/bonus effects. This establishes the core data structures and interfaces for the Armor Proficiency System (v0.16.2).

## Added

### Domain Layer - Enums (1 file)

| File                       | Description                                                                                    |
| -------------------------- | ---------------------------------------------------------------------------------------------- |
| `ArmorProficiencyLevel.cs` | Enum defining 4 proficiency levels: NonProficient (0), Proficient (1), Expert (2), Master (3). |

### Domain Layer - Value Objects (1 file)

| File                        | Description                                                                                                  |
| --------------------------- | ------------------------------------------------------------------------------------------------------------ |
| `ArmorProficiencyEffect.cs` | `readonly record struct` encapsulating armor effects per proficiency level. Factory methods with validation. |

### Application Layer - Interfaces (1 file)

| File                                 | Description                                                                                         |
| ------------------------------------ | --------------------------------------------------------------------------------------------------- |
| `IArmorProficiencyEffectProvider.cs` | Interface for armor proficiency effect data access. Methods for effects, modifiers, and validation. |

### Configuration (2 files)

| File                                                   | Description                                            |
| ------------------------------------------------------ | ------------------------------------------------------ |
| `config/armor-proficiency-effects.json`                | Effect definitions for all 4 armor proficiency levels. |
| `config/schemas/armor-proficiency-effects.schema.json` | JSON Schema draft-07 for configuration validation.     |

### Unit Tests (2 files)

| -------------------------------- | ---------------------------------------------------------------------- |
