# v0.16.2c Changelog - Archetype Armor Matrix

## Summary

Implements the **Archetype Armor Matrix** that defines which armor categories each archetype is proficient with and establishes Galdr/WITS interference rules for caster archetypes. This provides the foundation for archetype-specific armor usage and magic penalty calculation.

- Mystics use Galdr (blocked by heavy armor Faraday-cage effect)
- Adepts use WITS (penalized but not blocked by heavy armor)
- All archetypes start proficient with Light armor
- Specialized armor requires explicit training for all archetypes
