# Armor Proficiency System

## Overview

Armor proficiency determines a character's ability to wear armor effectively. Wearing armor without proficiency imposes severe penalties; achieving proficiency enables normal function.

### Proficiency Levels

| Proficiency Level | Effect |
| :--- | :--- |
| **Non-Proficient** | All penalties doubled; -2 to all attack rolls; cannot use special properties |
| **Proficient** | Normal penalties and function; full access to special properties |
| **Expert** | Penalties reduced by one tier (Heavy → Medium penalties) |
| **Master** | Penalties reduced by one tier; +1 Defense; armor maintenance time halved |

### Non-Proficiency Penalty Examples

| Situation | Normal Penalty | Non-Proficient Penalty |
| :--- | :--- | :--- |
| **Mystic in Medium Armor** | -1d10 Agility, +2 Stamina | -2d10 Agility, +4 Stamina |
| **Skirmisher in Heavy Armor** | -2d10 Agility, +5 Stamina | -4d10 Agility, +10 Stamina |
| **Mystic in Heavy Armor** | -2d10 Agility, +5 Stamina | -4d10 Agility, +10 Stamina, cannot cast Galdr |

## Proficiency Categories

| Category | Armor Types Included |
| :--- | :--- |
| **Light Armor** | Cloth, padded, leather, hide, studded leather |
| **Medium Armor** | Chain mail, scale mail, ring mail, brigandine, splint armor |
| **Heavy Armor** | Half plate, full plate, fortress armor |
| **Shields** | Bucklers, round shields, kite shields, tower shields |
| **Specialized** | Servitor shell, symbiotic carapace (requires additional training) |

## Archetype Proficiency Matrix

| Archetype | Proficient Armor | Non-Proficient Armor |
| :--- | :--- | :--- |
| **Warrior** | Light, Medium, Heavy, Shields | None |
| **Skirmisher** | Light, Medium | Heavy, Tower Shields |
| **Mystic** | Light only | Medium, Heavy (Medium/Heavy blocks Galdr channeling) |
| **Adept** | Light only | Medium, Heavy (Heavy impairs WITS-based analysis) |

> [!WARNING]
> **Galdr Channeling Interference**
> Medium and Heavy armor interfere with Aetheric energy flow, disrupting Galdr channeling. Mystics wearing **Medium armor** suffer **-2 to all Galdr checks**. Mystics wearing **Heavy armor** cannot cast Galdr abilities at all—the metal mass creates a Faraday-cage effect that blocks Aetheric manipulation.

## Proficiency Acquisition

| Method | Proficiency Gained |
| :--- | :--- |
| **Archetype Selection** | Starting proficiencies per archetype |
| **Specialization** | Additional proficiencies per specialization path |
| **Training (NPC)** | 100–400 PS cost; 2–6 weeks; one proficiency category |
| **Combat Experience** | 15 successful combats in non-proficient armor grants proficiency |
| **Progression Point** | Spend 3 PP to gain one armor proficiency category |
