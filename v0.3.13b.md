# Changelog: v0.3.13b - The Combat Simulator
**Release Date:** 2026-01-20

## Summary
This release implements the **Combat Simulator**, a headless simulation engine for balancing the game's combat math. Developers can now pit AI-controlled player archetypes against enemies in thousands of rapid-fire battles to determine win rates, turn counts, and resource drain metrics. The implementation spans the **Engine Layer** (simulation services, agent logic) and **Terminal Layer** (CLI tooling), enabling data-driven balance adjustments without manual playtesting.

## Code Implementation Details

### Models

#### SimMatchResult

public record SimMatchResult(
    string Winner,

    int PlayerMaxHp,
    int StaminaSpent,

#### SimBatchResult
```csharp
public record SimBatchResult(
    string Archetype,
    string EnemyId,
    int TotalMatches,
    double WinRate,
    double AvgTurns,
    double AvgHpRemaining,
    double AvgStaminaSpent
);
```

### Services

#### CombatSimulationService
**Methods:**
- `Task<SimBatchResult> RunBatchAsync(string archetypeStr, string enemyId, int count)`
  - **Behaviors:**
    - Parses archetype string to `ArchetypeType` (defaults to Warrior).
    - Uses `Parallel.ForEachAsync` to run matches concurrently.
    - Creates isolated `IServiceScope` for each match to ensure state purity.
    - Fallback to `und_draugr_01` if enemy ID is invalid.
    - Limits matches to 100 turns to prevent infinite loops.
    - Aggregates results into a `SimBatchResult`.

#### SimulatedPlayerAgent
**Methods:**
- `Task TakeTurnAsync(Character player, Enemy target)`
  - **Behaviors:**
    - **Survival Check:** (Planned) Checks if HP < 30% to use potions (currently placeholder).
    - **Aggressive Strategy:** If Stamina >= 40, performs `Heavy` attack to dump resources.
    - **Conservation Strategy:** If Stamina < 40, performs `Light` attack to conserve/regenerate.

### CLI Implementation (`Program.cs`)
- **Command:** `dotnet run -- --sim-combat --archetype=Warrior --enemy=und_draugr_01 --count=1000`
- **Output:**
  - Console summary: `Simulation complete. Win Rate: 98.5%`
  - Report file: `docs/balance/sim_combat_YYYYMMDD_HHmmss.md` containing full statistics.
