## Summary

Implements the command input panel with a text input field, command history navigation (up/down arrows), and tab completion with a popup showing suggestions. Integrates existing TUI services with the Avalonia GUI.

## Features

| Feature | Description |
|---------|-------------|
| Command prompt | Visual `>` prompt indicator |
| Text input | Monospace input with watermark |
| History navigation | Up/down arrows recall previous commands |
| History indicator | `[History ▲]` shown when navigating |
| Tab completion | Single match applies inline |
| Completion popup | Multiple matches show in dropdown |
| Popup navigation | Up/down to select, Enter to accept |
| Escape to cancel | Closes popup without selection |

---

## Key Bindings

| Key | Popup Hidden | Popup Visible |
|-----|--------------|---------------|
| Enter | Submit command | Accept selection |
| Up | Previous history | Select previous |
| Down | Next history | Select next |
| Tab | Trigger completion | - |
| Escape | - | Hide popup |
