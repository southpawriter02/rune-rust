# v0.15.6g - Investigation Command

**Release Date:** 2026-01-26

## Summary

Implements value objects and interfaces for **Investigation Command** with clue discovery and deduction mechanics.

## Added

### Enums (Domain)

| Enum                  | Values                                                                     |
| --------------------- | -------------------------------------------------------------------------- |
| `InvestigationTarget` | CrimeScene, Remains, Wreckage, Trail, Document                             |
| `ClueCategory`        | Identity, Cause, Method, Motive, Location, Time, Creature, Object, History |

### Value Objects (Domain)

| Value Object              | Purpose                                  |
| ------------------------- | ---------------------------------------- |
| `Clue`                    | Evidence discovered during investigation |
| `Deduction`               | Logical conclusion from combined clues   |
| `InvestigationContext`    | Ongoing investigation state              |
| `InvestigationResult`     | Complete investigation action results    |
| `InvestigationDifficulty` | Static DC reference values               |

### Interfaces (Application)

| Interface                     | Purpose                                          |
| ----------------------------- | ------------------------------------------------ |
| `IInvestigationService`       | Core investigation operations                    |
| `IPlayerClueTracker`          | Tracks discovered clues/deductions per character |
| `IClueRepository`             | Clue data access                                 |
| `IDeductionRepository`        | Deduction data access                            |
| `IInvestigationConfiguration` | Investigation time and settings                  |

## Key Features

### Clue VO

- Category-based organization for deduction matching
- Discovery DC, tags, weight system
- Source tracking (target type and ID)

### Deduction VO

- Required clue prerequisites
- Unlocks interactions and sets story flags
- `CanDeduce()` and `GetMissingClueCount()` helpers

### Investigation Time Costs

| Target      | Time   |
| ----------- | ------ |
| Crime Scene | 15 min |
| Remains     | 10 min |
| Wreckage    | 15 min |
| Trail       | 5 min  |
| Document    | 5 min  |
