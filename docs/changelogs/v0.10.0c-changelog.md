## Summary

Implements effect triggers (DoT/HoT), immunity checking, and cleanse/dispel mechanics.

---

## Added

### Domain Layer - Enums

| Enum | Values |
|------|--------|
| `TriggerTiming` | OnApply, OnTurnStart, OnTurnEnd, OnTick, OnExpire, OnRemove, OnDamaged, OnAttack |
| `TriggerActionType` | Damage, Heal, ApplyEffect, RemoveEffect, ModifyStat, Custom |

### Application Layer - DTOs

| DTO | Description |
|-----|-------------|
| `TickResult` | Aggregates DamageDealt, HealingDone, ExpiredEffects, TriggeredEffects |

### Application Layer - Events

| Event | Description |
|-------|-------------|
| `DoTDamageDealtEvent` | Raised after DoT damage |
| `HoTHealingDoneEvent` | Raised after HoT healing |
| `EffectsCleanedEvent` | Raised when debuffs cleansed |
| `EffectsDispelledEvent` | Raised when buffs dispelled |
| `CustomTriggerEvent` | Raised for custom triggers |

### Application Layer - Interface Extensions

| Method | Description |
|--------|-------------|
| `TickEffects(target, timing)` | Process triggers for turn events |
| `IsImmune(target, effectId)` | Check immunity before applying |
| `Cleanse(target, count?)` | Remove debuffs |
| `Dispel(target, count?)` | Remove buffs |
