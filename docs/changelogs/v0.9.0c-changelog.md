## Summary

Implements a TUI bell service providing console beep notifications for important events in terminal mode with 8 distinct patterns for different event types.

---

## Added

### Application Layer - Enums

| Enum | Description |
|------|-------------|
| `BellType` | 8 bell pattern types |

**BellType Values:**
- `Info` — Single short beep (800Hz, 100ms)
- `Warning` — Double beep (600Hz, 150ms × 2)
- `Error` — Long beep (400Hz, 500ms)
- `Combat` — Quick triple (1000Hz, 80ms × 3)
- `LevelUp` — Ascending tones (600→800→1000Hz)
- `QuestComplete` — Celebratory pattern
- `LowHealth` — Danger pattern (400Hz × 2)
- `Notification` — Simple beep (700Hz, 120ms)

### Application Layer - Interfaces

| Interface | Description |
|-----------|-------------|
| `ITuiBellService` | TUI bell operations |

**ITuiBellService Methods:**
- `Bell(BellType)` — Play predefined pattern
- `BellCustom(frequency, durationMs)` — Custom beep
- `IsEnabled` — Enabled state property
- `SetEnabled(bool)` — Toggle bells

### Infrastructure Layer - Services

| Class | Description |
|-------|-------------|
| `TuiBellService` | Bell implementation |
| `BellPattern` | Pattern definition record |
| `BellNote` | Note definition record |

**Cross-Platform Support:**
- Windows: Native `Console.Beep(freq, duration)`
- Unix/Mac: Fallback to `'\a'` terminal bell
