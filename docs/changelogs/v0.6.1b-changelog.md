# v0.6.1b Changelog: ASCII Room Renderer

**Version:** 0.6.1b

**Status:** ✅ Complete

## Summary

Implements ASCII art room visualization with symbol-based coloring, entity overlays, and legends. Rooms can have custom art that displays in the main content panel with dynamic elements like player/monster positions.

## Key Features

| Feature | Description |
|---------|-------------|
| **Custom Room Art** | Per-room ASCII art definitions |
| **Symbol Colors** | Colored symbols via ConsoleColor mapping |
| **Entity Overlay** | Dynamic player/monster position overlay |
| **Legend Rendering** | Symbol explanations displayed with art |
| **Art Scaling** | Centering/truncation for panel width |
| **Default Fallback** | Generic art for undefined rooms |
